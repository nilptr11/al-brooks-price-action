# 市场周期（Market Cycle）

> **状态：Core / Index**
>
> 本页说明市场状态专题的职责和文件入口，不把状态标签直接解释为交易许可。

## 职责

本目录负责区分 trend 与 trading range，以及 breakout、tight channel、broad channel、breakout mode 和 climax 如何描述状态及其转换。

Market Cycle 回答“市场现在主要怎样运动”；具体位置与控制权属于 [Context](../02_context/README.md)，入场命题属于 [Setups](../05_setups/README.md)。

## 文件

1. [`00_market_cycle.md`](00_market_cycle.md)：基础层级、连续谱和动态更新。
2. [`01_trends_and_channels.md`](01_trends_and_channels.md)：趋势、紧通道、宽通道和回调质量。
3. [`02_trading_ranges.md`](02_trading_ranges.md)：交易区间、紧区间、宽区间及双边行为。
4. [`03_breakouts_and_breakout_mode.md`](03_breakouts_and_breakout_mode.md)：突破事件、突破质量与 breakout mode。
5. [`04_climax_and_transition.md`](04_climax_and_transition.md)：高潮式推进、已确认 climax 和状态转换。

概念总图和权威定义归属见[核心框架](../README.md)。
