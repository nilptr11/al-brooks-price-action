# 市场周期专题导航（Market Cycle）

> **状态：Core / Index**
>
> 本页说明市场状态专题的职责和文件入口，不把状态标签直接解释为交易许可。

## 职责

本目录先区分 trend 与 trading range 两种基础状态，再说明获接受的 breakout phase（spike）、tight channel 和 broad channel 如何描述 trend 的阶段；breakout mode、climactic move 与 transition evidence 则作为可叠加在基础状态上的跨层信息。本目录也区分最低 breakout event / attempt 与趋势中的 breakout phase，避免用同一词混写不同粒度。

Market Cycle 回答“市场现在主要怎样运动”；具体位置与控制权属于 [Context](../02_context/README.md)，入场命题属于 [Setups](../05_setups/README.md)。

## 文件

1. [`00_market_cycle.md`](00_market_cycle.md)：基础状态、趋势阶段、跨层信息、连续谱和动态更新。
2. [`01_trends_and_channels.md`](01_trends_and_channels.md)：趋势中的 breakout phase（spike）、紧通道、宽通道和回调质量。
3. [`02_trading_ranges.md`](02_trading_ranges.md)：交易区间、紧区间、宽区间及双边行为。
4. [`03_breakouts_and_breakout_mode.md`](03_breakouts_and_breakout_mode.md)：breakout event / attempt、接受结果、breakout phase 与 breakout mode 的粒度边界。
5. [`04_climax_and_transition.md`](04_climax_and_transition.md)：高潮式推进、已确认 climax 和状态转换。

概念总图和权威定义归属见[核心框架](../README.md)。
