> **状态：Core / Index**

## 职责

本目录定义 Brooks 常用 K 线角色和形态词汇的最低边界。Pattern 负责描述价格行为；它的交易含义必须回到 Market Cycle、Context、follow-through 和目标空间。

Pattern 与 Setup 的边界由[形态只是语言](00_patterns_are_language.md)和[什么是 Setup](../05_setups/00_what_is_a_setup.md)共同衔接，具体交易命题不在本目录重复展开。
