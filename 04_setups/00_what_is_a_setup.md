# 什么是 Setup

## Brooks 语境中的定义

Setup 不是形态名称，而是完整交易想法。一个 setup 至少包含：

- Market cycle。
- Context 和 location。
- 方向理由。
- 触发点。
- 失效点。
- 目标。
- 管理方式。
- Trader's Equation。

## Setup 和 Pattern 的区别

Pattern 描述市场行为，例如 H2、wedge、double top。Setup 说明是否值得交易、怎么入场、错了在哪里、目标在哪里、如何管理。

同一个 pattern 可以支持不同 setup，也可以完全没有交易价值。

## Setup 检查表

看到可能交易点时，按顺序问：

1. 当前环境支持 stop entry、limit entry，还是等待？
2. 价格是否在好位置？
3. 触发点在哪里？
4. 触发后谁会被困？
5. 合理止损在哪里？
6. 第一目标在哪里？
7. 这笔交易是 scalp 还是 swing？
8. 概率、风险、回报和成本是否合算？

如果答案需要大量解释，通常说明 setup 不够清晰。
