# 48H：尾盘支撑阻力测试、反转与“80% 规则”

## 内容结构

1. **日中末段测试关键磁点**（PDF 物理页 1020–1021）
   - 交易区间日若开盘价位于全天区间中间三分之一，后半日常回测开盘价。
   - 关键磁点也可以是前高前低、趋势线或测量目标；测试后的反应决定最后三分之一是继续横盘、向上趋势还是向下趋势。
2. **晚盘强突破可能是陷阱**（PDF 物理页 1022–1025）
   - 价格未能突破阻力后，可能快速向下测试支撑；即便出现两根强空头趋势 K 线，该下跌仍可能只是卖出真空。
   - 支撑测试后的反转可启动尾盘相反趋势。激进交易者在关键位附近用限价单，保守交易者等待 H2/更高低点、连续强阳线或跟进后买入。
3. **强趋势仍可能只是交易区间腿**（PDF 物理页 1026）
   - 当开盘价位于区间中部，强反弹到测量目标、大整数、三角形或最终旗形后，尾盘可能完全回落并收近开盘，最终在日线上形成十字星。
4. **早入场、晚确认与止损结构**（PDF 物理页 1027–1028）
   - 反转 K 线下方的早期入场价格较好但概率较低；等待第三、第四根连续空头 K 线收低，概率更高但止损更远。
   - 窄止损被扫后再次入场，或用小仓位宽止损并在更高处加仓，都是可行但风险分布不同的方案。
5. **强突破状态下的“80% 规则”**（PDF 物理页 1030）
   - 市场具有惯性；一旦确认强空头突破，课程称约 80% 的反转尝试会失败，因此只寻找做空，直到出现主要趋势反转或强多头突破。
   - 同页把“可能”定义为至少约 60% 的主观置信度，并建议即使只用小仓位也应参与高概率方向。
6. **晚盘双底/双顶启动最后趋势**（PDF 物理页 1031）
   - 日中弱趋势若只是交易区间一条腿，测试当日低点或早前低点后的双底、更高低点主要趋势反转和强多头 K 线，可启动收盘前趋势。

## 核心思想

- 日中交易区间的重要功能之一是把价格带到关键支撑阻力，测试结果为尾盘重新定价。
- 尾盘关键位测试后的三类结果是：围绕关键位继续小区间、向上突破并收高、向下突破并收低；方向必须由反应和跟进确认。
- 强 K 线不必等于真突破。在交易区间背景、关键磁点附近，强烈单边移动可能只是流动性真空和陷阱。
- 越晚等待确认，方向概率通常越高，但入场价格、止损距离和剩余交易时间更差；仓位必须随风险距离缩小。
- 市场状态一旦从交易区间切换到强突破，反转思维应停止；主要趋势反转或强反向突破出现前，顺势优先。
- 晚盘反转形态的意义来自位置、买卖压力和强跟进，而非“双底/双顶”名称本身。

## 重复与递进

- **合理重复：交易区间日尾盘回测开盘价。** 47D、48C 已出现，本节把它置于“中段测试—尾盘三种结果”的完整流程中，是最适合作为该思想主定义的位置。
- **合理重复：强移动可能是交易区间陷阱。** 与第 47 讲的第二腿陷阱和 48G 的区间腿判断一致，本节新增尾盘关键位与卖出真空条件。
- **合理重复：早入场低概率高回报，晚入场高概率低回报。** 48G 已提出，本节用第三/第四根趋势 K 线及远止损展示风险变化，属于执行细化。
- **合理重复：宽止损加仓或窄止损再入场。** 与 47C–47D 相同，本节只列两种结构，不应再次把宽止损描写为更安全。
- **合理重复：TBTL、最终旗形、楔形和测量移动。** 物理页 1026–1027 是尾盘反转场景引用，基础定义应交叉链接。
- **应区分而非合并：“80% 规则”有两个不同对象。** 第 47 讲称交易区间中约 80% 的突破会失败；本节称强突破后约 80% 的反转尝试会失败。两者都表达当前状态的惯性，但条件完全相反，必须分别命名。

## 可能矛盾或边界

- **开盘价回测倾向只适用于已显现交易区间、且开盘位于中部。** 若趋势日已经形成或开盘靠近区间边缘，磁吸作用和尾盘路径会不同。
- **“测试后的反应决定尾盘方向”带有回看性。** 实时必须用突破、跟进和失败信号定义反应，不能等收盘后再证明该测试重要。
- **晚盘强突破既可能是陷阱，也可能开启趋势。** 物理页 1022 与 1030 表面相反；区分条件是突破前是否仍为交易区间、是否位于磁点，以及突破后是否持续强劲并获得跟进。
- **5 分钟图交易区间与日线趋势 K 线可以同时成立。** PDF 物理页 1022–1023 的收高使日线显强，只说明周期聚合结果不同，不表示日内过程一直是趋势。
- **“通常反转并测试另一侧”不能保证完成目标。** 剩余时间、日内波幅是否已扩张、阻力失败强度都会影响是否到达支撑。
- **晚入场“概率更高”不等于期望更好。** 止损更远、目标剩余空间更小，必须缩小仓位并重新计算盈亏比；材料没有提供统一最优方案。
- **宽止损加仓和窄止损再入场不可只比较胜率。** 前者尾部风险更大，后者承担多次小亏与滑点；两者都需预设总风险上限。
- **本节“80%”与第 47 讲“80%”不能共用一句无条件规则。** 交易区间惯性反对突破，强突破惯性反对反转；先正确识别状态才可能应用。
- **“必须至少做小仓位”是课程的执行主张，不是逻辑必然。** 不交易仍是合法选择，尤其当止损距离、尾盘时间或个人计划不匹配时。
- **“probably 至少 60%”是作者词汇约定。** 它不是自然语言中通用定义，也未给统计校准方法，仓库应标明为课程内部用语。
- **尾盘双底后持有至当日高点/收高是案例结果。** 不能把目标写成每个晚盘双底的默认保证。

- PDF 物理页 1021 中文抽取遗漏了第一种结果“余下时间围绕开盘价维持小区间”，三种结果不能只剩多头与空头两类。
- PDF 物理页 1022 的完整中英文对照被拆到物理页 1023；`but ls` 是 `bulls` 的 OCR 损坏。
- PDF 物理页 1023 中文“随后的上涨反转是最后的开始”遗漏 `final third`，应为“开启当天最后三分之一阶段”。
- PDF 物理页 1024 的激进限价入场说明主要落在英文图注，中文正文未完整抽取；物理页 1025 补出了该段。
- PDF 物理页 1027 的图形箭头和 `3 2` 计数无法从纯文本可靠定位；可以确认早期反转入场与第三/第四根确认入场的对照。

- PDF 物理页 1031 的“并收盘于当日开盘价和高点”意为收盘回到开盘价附近且位于当日高位，中文并列容易读成两个价格相同。
- PDF 物理页 1018–1019、1029、1032–1033 主要为封面、目录、分节与结束页，没有额外独立论点。
