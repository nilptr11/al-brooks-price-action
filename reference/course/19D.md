# 19D：收盘追随者、入场价磁铁与突破测试

来源：`SRC-COURSE-01-36/19D.txt`，对应 PDF 物理页 1680–1700。

## 内容结构

1. 说明强趋势中的 Buy The Close / Sell The Close 参与者如何在强收盘追随趋势（PDF p1682、p1686）。
2. 讲解回调过强后，这些参与者如何把市场重分类为交易区间、分批建仓并在原入场价退出（PDF p1683–1691）。
3. 区分失败交易中原入场价形成的反向水平，与成功突破回测中入场价形成的顺势水平（PDF p1691、p1693–1696）。
4. 将突破测试扩展到入场 K 线和信号 K 线的高低点（PDF p1697–1698）。

## 核心思想

- 强多头趋势中，Buy The Close 多头在强阳线收盘买入，押注即使先有回调，价格最终仍会更高；强空头趋势中的 Sell The Close 空头则在强阴线收盘卖出，逻辑完全镜像（PDF p1682、p1686）。
- 如果回调中的反向压力过强，原顺势参与者会感到失望，并把市场从趋势重新判断为交易区间。此时不再单纯追随，而是在更有利价格分批增加仓位，等待价格恢复到原入场附近退出（PDF p1683–1684、p1687–1688）。
- 典型分批退出路径是：第一笔在原收盘价入场，第二笔在更低处买入或更高处卖出；价格回到第一笔入场价时，第一笔保本、第二笔获利，然后整体退出。这使原入场价成为集中退出的磁铁（PDF p1684、p1688、p1691、p1694）。
- 多头趋势结束并转为区间后，最高强阳线收盘或第一笔买入价常成为阻力，因为失望多头在反弹回该价位时卖出退出，空头也可能在那里新开仓，随后容易形成区间内下跌腿（PDF p1685）。
- 空头趋势结束并转为区间后，最低强阴线收盘或第一笔卖出价常成为支撑，因为失望空头在回落到该价位时买回退出，多头也在那里买入，随后容易形成上涨腿（PDF p1689–1690）。
- 若回调甚至未能触及最低收盘价，说明空头回补和新多头买入提前出现，买压较强，市场更可能形成第二段上涨（PDF p1689）。
- 入场价的支撑阻力角色取决于交易是否成功。失败多头在原买入价卖出，使其成为阻力；失败空头在原卖出价买回，使其成为支撑（PDF p1691、p1694）。
- 成功多头突破后的回测则相反：原买入价附近有顺势多头再次买入，成为支撑；成功空头突破后的回测中，原卖出价附近有空头再次卖出，成为阻力（PDF p1693、p1695–1696）。
- 突破测试不只参考精确成交价。多头入场 K 线和信号 K 线的低点可成为支撑，价格测试后向上反转可触发买入；空头入场 K 线和信号 K 线的高点可成为阻力，测试后向下反转可触发卖出（PDF p1697–1698）。
- 突破回测处可以有两类顺势入场：提前在预期水平挂限价单，或等待价格从水平反转后用停止单确认。两者分别交换入场价格与确认程度（PDF p1693、p1695–1696）。

## 重复与递进

- p1682–1685 与 p1686–1690 是多头、空头完整镜像：强收盘追随、回调失望、分批、原价退出、最高/最低收盘角色转换。教学重复合理，知识条目可合并。
- p1683–1684、p1687–1688 都在解释趋势转区间后分批交易；后页增加了“第一笔保本、第二笔获利、全部退出”的目标，属于递进。
- p1685、p1690 分别解释最高收盘阻力和最低收盘支撑，是同一订单机制的方向对称。
- p1691 与 p1694 讲失败多空交易的原入场价角色；p1693 与 p1695 讲成功突破回测的入场价角色。四页必须成组保留，否则容易得到“买入价永远是支撑”之类错误规则。
- p1695–1696 内容高度相近，后者把案例明确为趋势线突破后的回调，可合并。
- p1697–1698 是入场/信号 K 线低点与高点的多空镜像。
- 本讲把 19B–19C 中“入场价和 K 线高低点是支撑阻力”的清单展开成订单机制，新增了失败交易与成功突破时角色相反这一关键条件。

## 可能矛盾或边界

- 强趋势中在收盘追随，与趋势转区间后低买高卖并不矛盾；切换条件是回调反向压力已经强到破坏原先的单边预期，而不是任何普通回调。
- “回调令人失望”是相对于顺势持仓者而言：多头看到过强卖压、空头看到过强买压。中文若不写明观察方，容易误解为回调太弱。
- 分批后回到第一入场价能使第一笔保本、第二笔获利，但前提是第二笔确实成交、仓位和成本合理，且市场能够回测。价格继续单边不返会扩大风险。
- 本讲的分批策略只在交易区间或弱趋势、原交易前提仍有效时成立。若出现强突破否定前提，不应为了等保本而继续加仓。
- 原买入价既可成为阻力也可成为支撑，原卖出价也一样。决定因素是该价位聚集的是失望者退出，还是成功突破后的顺势者再次入场。
- 最高/最低收盘是候选区域而非精确保证。不同参与者成交价、手续费和分批方式不同，订单会分布在附近。
- “常导致一段上涨/下跌”描述区间内订单倾向，不表示一定形成完整波段或趋势。
- 用停止单等待反转确认会得到较差价格；用限价单提前入场则可能在突破尚未停止时逆势成交。两种方式都需要结构失效止损。

## 翻译或抽取问题

- PDF p1683 标题称“在回调期间加仓”，英文正文更具体地说在多头重新恢复时增加仓位；不能确定所有加仓都发生在回调继续下跌过程中。
- PDF p1685 中文把多头退出的限价卖单写成“限价空单”，应区分平多卖出与新建空仓，尽管两者都会贡献卖压。
- PDF p1687 的 `1–2x size of scalp` 是案例中的分批距离量级，缺少品种和具体最小短线定义，不宜形成通用参数。
- PDF p1689 中文残留 `buy`；含义是空头回补和多头新买共同使下跌未到最低收盘，并支持第二段上涨。
- PDF p1690 中文把买回空仓的限价单写成“限价多单”，应区分回补与新开多仓。
- PDF p1693 的 `stop on reversal up` 是停止单确认买入，不是保护性止损。
- PDF p1695 中文“使用限价单在多头退出的位置平仓”方向错误；英文是空头在分批多头退出的位置用限价单卖出入场。
- PDF p1696 同样把 `Sell with limit order` 混入“平仓”语义，应还原为新建空头，交易对手可能是退出的多头。
- PDF p1697–1698 标题中文加入“做多多单/做空空单”造成冗余；核心是入场 K 线和信号 K 线的低点/高点作为测试水平。
- PDF p1680–1681、p1692、p1699–1700 为封面、目录、分节或结束页，不提供独立论据。
