# 46E：宽幅空头通道的卖出区、50% 回调与最终区间化

来源：`SRC-COURSE-37-52/46E.txt`，对应第二册 PDF 物理页 774–796。

## 内容结构

1. 定义强空头突破/紧密通道与宽幅通道中的卖出区，对比回调卖出和新低追卖（PDF p776–779）。
2. 讨论前低下方止盈、买入真空、结构止损和 50% 回调卖出（PDF p780–786）。
3. 把长期空头趋势内区间视作空头旗形，同时说明下破常失败和通道最终区间化/转多（PDF p787–794）。

## 核心思想

- 强空头突破或紧密空头通道中，卖出区约在前一下跌腿中点，可在 50% 回调限价卖或等向下反转确认（PDF p776）。
- 当宽幅空头通道越来越像区间、反弹超过 50% 并越过重要高点时，空头优先在前一下跌腿顶部三分之一寻找卖出区；可在约 60%–70% 回撤限价卖，或等向下反转（PDF p777）。
- 因长期仍是空头趋势，任何合理位置都可卖，只要止损在当前空头腿顶部上方；但宽幅通道像区间，反弹卖出通常比前低下破追卖更容易获利（PDF p778–779）。
- 前低下方是空头止盈区，也是多头 scalp 买入区；基础循环是反弹卖出，在接近/略破新低时止盈（PDF p780）。
- 宽幅通道预期深反弹，结构止损可放最近主要 LH 上方。部分空头在通道顶部/均线附近的大阳线收盘卖、押注多头上破失败；更保守者等待接近上一 LH/双顶（PDF p781）。
- 因卖方刻意等待上半部，下半部会出现卖盘真空并造成看似强的多头突破。到达卖出区后空头积极卖出，若迅速压倒多头并形成新 LH，该突破才被确认是 buy vacuum 而非新多头趋势（PDF p782）。
- 有多个合理止损时，材料再次声称紧止损亏得少但更频繁、宽止损亏得多但较少，约十笔后利润大致相同；若退出而空头通道仍有效，可在向下反转重新卖出（PDF p783）。
- 50% 回调卖出的简化方程式是：若止损在空头腿起点、目标在前低，则风险约等于回报；空头趋势先验给约 60% 到目标、40% 到止损，因此期望为正（PDF p785）。
- 即使此前只是较小的向下反转，只要它从通道顶部足够强，也可在该腿 50% 回调限价卖或等向下反转 stop-entry 卖；止盈在新低、向上反转或时段结束（PDF p786）。
- 一个十日五分钟宽幅空头通道在 60 分钟图可为紧密通道，内部反复包含交易区间（PDF p787）。
- 空头趋势内每个交易区间都会出现扩张三角、楔形、双底、HL MTR 等筑底尝试；同一形态可有多个标签，关键是多头在寻找底部（PDF p788）。
- 当底部失败，多头止损/放弃，空头卖下破并期待测量移动；由此区间又可被重新解释为空头旗形，任何反弹在趋势持续时也可视作空头旗形（PDF p789–791）。
- 区间处于 breakout mode，局部多空概率接近 50%；只要长期仍形成 LH，高周期仍有轻微空头优势，五分钟日内多空获利难度可近似（PDF p792）。
- 通道下方空头突破被材料赋予约 75% 在最高显示该形态的周期约五根内回升、约 25% 成功并进入新空头通道/向下测量移动。确认成功后按普通突破做空并 swing（PDF p793）。
- 所有通道最终结束。材料把路径简化为：约 25% 强空头下破并恢复新通道，约 75% 演变成大交易区间或多头趋势；形成主要 HH 后停止按空头趋势卖，继续形成 HL 则进入多头通道（PDF p794）。

## 重复与递进

- 46E 是 45E 的逐页多空镜像：60%–70% 区域、回调优于突破、前低/前高止盈、真空、十笔止损说法、50% 回调方程式、区间旗形、25%/75% 最终路径均方向翻转。
- p776–777 在同一讲中区分强突破/紧密通道的 50% 卖区与宽幅/区间的顶部三分之一（约 60%–70%），是比镜像 45E 更清楚的上下文分层。
- p780–782 重复深回调和买入真空，新增“等待卖出导致下半部缺卖方”的过程解释。
- p783 再次重复十笔后紧/宽止损利润相近，与 43D 的 100 笔冲突；这是同一待核簇的第四次出现。
- p788–792 连续多页表达“区间同时含筑底和空头旗形”，核心是形态多标签与方向取决于突破，不宜拆成多个绝对定义。
- p793–794 用相同 25%/75% 先讲下破是否成功，再讲长期最终路径；事件窗口不同，不能当重复统计相互验证。
- 45A–E 与 46A–E 十段整体构成第二个巨大多空镜像簇；材料分析应完整保留，仓库正文应以方向中性宽幅通道模型承载。

## 可能矛盾或边界

- p776 的中点卖出区与 p777 的顶部三分之一不是矛盾：前者用于强突破/紧密通道浅回调，后者用于宽幅/区间深回调；知识库需明确背景。
- p777 的“顶部三分之一”对应从低点回撤约 66%–100%，60%–70% 大致在边界附近，并非严格同一区间。
- p782 又把卖出区括注为“顶部 2/3”，比 p777 的顶部 1/3 宽得多，是明确口径不一致，镜像了 45E 的底部 1/3 vs 2/3。
- “任何时间任何理由都可卖”与“不要在前低下方突破卖”可协调为可行与优选之分，但绝对措辞会误导。
- 大阳线收盘卖出只有在卖出区/长期空头背景且多头无跟进时成立；强上破主要 LH 并跟进时不能继续称 buy vacuum。
- p783 的十笔利润趋同说法仍缺样本、成本与策略，且与 100 笔镜像页冲突，不可作为统计规律。
- p785 的 50% 处风险等于回报仅在止损为下跌腿起点、目标为前低、无成本时成立；真实结构会改变比例。
- 60% 到目标/40% 到止损是未给样本的课程先验，弱下跌腿或通道已退化时不能照搬。
- “每个区间都含筑底形态/空头旗形”若允许多重标签，接近定义性事实；其预测价值来自实际突破与跟进，而非标签数量。
- p789“每次空头突破都是失败底部”是事后重分类：成功向上反转的底部不会被叫失败，存在循环性。
- 局部 BO mode 近 50/50 与高周期空头略优不冲突，必须标注时间周期和目标距离。
- p793 的 75% 指最高显示形态周期约五根内回升，但图例注明九根后回升；这不推翻概率命题，却说明“五根”不是硬截止。应保留为直接边界例。
- p793 的成功 25% 只有在强下破跟进确认后才可按任意理由卖，不能事前假定每次尝试属于成功组。
- p794 的 75% 是最终成区间/多头趋势，与 p793 的五根内反转不是同一事件。
- 单个 HH/HL 在区间很常见；p794 用它们终止空头趋势过度简化，应限定为主要摆点和持续跟进。
- `Bear channel is Bull Flag` 在 p794 语境突兀，可能指通道破坏后成为更高周期多头旗形，不能直接认定所有空头通道同时是多头旗形。

## 翻译或抽取问题

- PDF p781 的 `OT` 应为 `DT`（双顶）OCR 错误；中文已译成双重顶。
- PDF p782 `top 2/3 rd5` 中 `5` 是 OCR 噪声，但“顶部 2/3”本身与 p777 冲突，应原样保留。
- PDF p783 图内 Major/Minor LH 与 PB 的对应止损位置依赖图，纯文本只能确认多个合理止损候选。
- PDF p785 的 60/40 是先到目标或止损的概率，不应译成普遍“盈利/亏损概率”而脱离目标定义。
- PDF p788 `HST` 在筑底语境可能为倒头肩或 head-and-shoulders top 标签混用；中文直译“头肩顶”与筑底主题冲突，需以后核图。
- PDF p789 中文标题“每一次空头突破都是失败的底部”是对英文图示的概括，不是英文标题直译。
- PDF p793 `Negative Gap` 译为反向缺口；示例明确九根后反转而非约五根。
- PDF p794 的 Spike and Channel Bull/Bear Trend 图注顺序与正文路径需结合图，纯文本无法判定两例对应位置。
- PDF p796 将 46E OCR 为 `A6E`。
- PDF p774–775、p784、p795–796 为封面、目录、分节或结束页，没有新增独立论据。
