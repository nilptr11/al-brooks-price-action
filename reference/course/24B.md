# 24B：不完美楔形、三推的功能定义与可交易条件

来源：`SRC-COURSE-01-36/24B.txt`，对应 PDF 物理页 1989–2013。

## 内容结构

1. 说明楔形很少完美，推动可由单根 K 线构成，也可在低周期才显出内部回调（PDF p1990–1992）。
2. 将旗形、通道和交易区间中的三推结构统一为楔形功能，并强调推动数比外形重要（PDF p1993–2000）。
3. 展开截短楔形、未触及通道线的第三推及失败双顶底等变体（PDF p2002–2004）。
4. 限定可交易楔形仍需相对强的推动和合理反转环境，不能只数到三（PDF p2005–2008）。
5. 说明楔形可与三角形、三重顶底、紧密交易区间重叠，并比较提前反转入场与突破后确认入场（PDF p2009–2011）。

## 核心思想

- 理想楔形并不常见。推动可能只有一根趋势 K 线或十字星，也可能有四段而非严格三段；当前周期看不到明显回调时，课程允许推断低周期存在三推结构（PDF p1991–1992）。
- 形态越接近理想楔形——三次清楚推动、反转可辨、环境支持且边界较协调——课程认为盈利反转概率越高；不完美只是降低可信度，不必自动取消（PDF p1991、p1995）。
- 楔形旗形内部的每一推可以很小：楔形多头旗形的空头腿可由一两根阴线构成，随后阳线是对该次空头突破的回调；楔形空头旗形则完全镜像（PDF p1992）。
- 旗形本身是小趋势或交易区间，因此多头旗形中的小空头趋势可结束于楔形底，空头旗形中的小多头趋势可结束于楔形顶。一个结构可同时叫旗形和楔形，职责取决于所描述的尺度（PDF p1993）。
- 对宽松楔形而言，推动数量比几何外观更重要。第二推可以越过连接第一、第三推的线，第一次反转后也可能创新高/低而不是形成标准 LH/HL（PDF p1994）。
- 课程把任何清楚的三次同向尝试都视为具有楔形功能，因此大多数运行到第三推的通道都可看作某类楔形；许多急速加通道趋势也以这种三推衰竭结束（PDF p1995–1998）。
- 实时识别流程是：出现两次反转后先画通道线，观察下一次触线、略微超越或未触及是否结束第三推。第三次尝试后出现反向信号，会增加反向摆动的机会（PDF p1997–1998）。
- 通道线只是组织三推的工具，不同画法可选取不同 K 线和推动。若信号 K 线方向不利、通道仍紧，课程建议等待第二次入场或明确突破，而不是仅因计数完成立刻交易（PDF p1997、p2000）。
- 三到四次突破尝试失败可以同时被描述为不规则楔形、扩张三角形或失败突破。名称不重要的可执行部分是：原方向多次不能获得接受，随后反向突破和跟进（PDF p1999）。
- 第三推不必达到第二推极值或通道边界。若交易者预期再创新低会完成楔形底，可能提前买入，空头也在前低上方回补，使第三次下探形成更高低点；课程称其为截短楔形底或 undershoot（PDF p2002–2003）。
- 失败双顶上破后再向下反转，会形成第三次向上尝试；失败双底下破后再向上反转，会形成第三次向下尝试。两者可按楔形功能交易，也可更自然地称为三重顶底、三角形或紧密交易区间（PDF p2004、p2009–2010）。
- “三推”只给出宽泛的功能分类，并不足以成为高质量交易。推动必须相对周围 K 线和腿足够明显，且所处环境应支持反转；否则横盘比趋势反转更可能（PDF p2005–2006）。
- 交易区间里的腿常以三次失败尝试结束，只足以产生次要反转和短线交易。若三推很弱、都在相近价位且没有强突破，把它称为三重顶底或紧密交易区间通常更清楚（PDF p2005、p2008–2010）。
- 对失败双顶形成的楔形顶，可在反转信号处提前卖出；等待大型空头趋势 K 线跌破区间并收在过去约四十根低点下方，盈利概率提高，但入场更低、止损更远、每单位价格风险更大（PDF p2011）。

## 重复与递进

- p1991–1995 多次重复“楔形很少完美、不同人/电脑会选不同腿、外形不如三推重要”；这是本讲的主旨，但正文可压缩成一次定义和一组反例。
- p1992–1993 将第 24A 的“楔形既可旗形也可 MTR”细化到一两根 K 线组成的小腿，是尺度层面的递进。
- p1995–1998 从“多数通道是楔形”递进到实时流程：两次反转画线，第三次触线/超越/未触及后观察反转信号。
- p1999–2000 用不同画线和不同名称再次说明形态标签不唯一，新增执行边界是坏信号或紧密通道时等待第二次入场。
- p2002–2003 是同一截短楔形底的两页解释，后一页补充提前买盘与空头回补的订单叙事，可合并。
- p2004 与 p2009–2011 都用失败双顶底产生第三推，后者补充替代名称和确认后入场的风险交换，内容高度重叠但有递进。
- p2005 与 p2006 是同一段核心限制的重复文本：三推不足，仍须推动相对强且反转环境合理。
- p2009 与 p2010 是同一页被拆分的互补抽取，p2010 补齐替代名称和失败双顶/底定义，不应算两个独立论据。
- p2011 重复 22B、23B 的“确认后概率更高，但止损更远、剩余回报较小”，在楔形案例中没有新增通用风险原则。

## 可能矛盾或边界

- p1995 明说“任何三推形态都符合条件”，p2005–2006 又说“三次推动还不够”。这是本讲最直接的表述张力。较合理的分层是：任何三推都可归入宽泛的楔形功能；只有推动清楚、相对强且环境合理时，才构成值得交易的楔形反转。
- “大多数通道都是楔形”只在通道已经出现第三推时成立；刚形成、只有一两推的通道不能预先确认为完成楔形（PDF p1997–1998）。
- 允许单根 K 线算一推、允许四推、允许无回调并推断低周期、允许不收敛和未触线，使定义极宽。它适合描述市场多次尝试，却难以单凭形态名称提供稳定概率。
- “低周期一定存在楔形”是从当前周期紧密走势作出的推断，不经低周期文本/图表无法验证。若定义允许不可见内部结构，计数会带有较强主观性。
- p1995 所称“越接近理想，盈利反转概率越高”与 p2005 的环境条件方向一致，但没有定义“理想程度”的可重复评分，不能据此精确排序交易。
- 第三推形成 undershoot 表示未达到通道线或前低，不是向下穿越。它可反映提前买盘，也可能仅是随机回调；仍需反转信号和向上突破确认。
- p2002 说“三次空头突破尝试失败”时，第三推本身却未必真正突破前低。这里的 breakout attempt 应理解为向下尝试，不应都写成已经发生的突破。
- p2005 把相近价位的三推改称三重顶底/TTR，p2009–2010 又说它们功能上仍是楔形。两者是命名优先级差异：功能可相同，但文档应选择最清楚、最常用的主标签。
- “市场/多空双方认为价格过高”“多头放弃”是解释性叙事；可观察证据是三到四次上破失败、反向趋势 K 线和跟进，不能把参与者一致意见当作事实。
- p2007 的案例仍在宽幅多头通道，三推后买压增加、向上突破更可能，说明三推并不天然预测反向大趋势；更大级别环境可能让楔形只结束交易区间内的一腿。
- 交易区间三推常只产生次要反转，与 24A 的 75% 至少小反转相容；不能将其写成波段或趋势反转成功率。
- 确认后“风险更大”仍指止损价格距离，不必等于账户金额风险更大；仓位缩小可以保持总风险不变。

## 翻译或抽取问题

- PDF p1991 的推动编号和多个不同楔形案例依赖图中位置，纯文本无法判断哪一个是三推、四推或单 K 线腿。
- PDF p1992 中文标题“小型楔形：通常伴随次要突破”没有对应的完整英文标题，可能是译制添加或从图下注释归纳，不宜作为原文定律。
- PDF p1994 的“第二推越过第一与第三推连线”需要图形才能判断是 overshoot 还是线条选择差异。
- PDF p1997 的英文是 `Undershoot`，中文译成“下穿”方向相反；这里应为第三推未触及/未到达通道线或先前极值，而非跌穿。
- PDF p2000 的 `Micro OT` 很可能是 `Micro DT` 的 OCR 错误；H1/H2/H3 和买点位置需要图示。
- PDF p2002–2003 标题与正文的 `undershoot` 再次被中文译为“下穿”，应统一修正为“未触线、欠冲”或保留英文并解释。
- PDF p2003 的“多头和空头都确信新低将形成楔形底”属于讲解叙事；抽取文本无法证明双方都持同一预期。
- PDF p2004 中文使用“双重顶/底”，仓库术语若统一为“双顶/双底”应避免无意义变体。
- PDF p2009 的英文和中文分布在 p2009–2010 两个物理页，需合并读取；单看 p2009 会漏掉“替代术语更恰当”的关键限制。
- PDF p2011 的过去四十根低点、区间突破和两个卖点依赖图形，纯文本只能确认条件，无法恢复精确价格位置。
- PDF p1989–1990、p1996、p2001、p2012–2013 为封面、目录、分节或结束页，不提供新增独立论据。
