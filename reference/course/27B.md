# 27B：头肩形的区间本质、颈线失败与止损后的状态重置

来源：`SRC-COURSE-01-36/27B.txt`，对应 PDF 物理页 2178–2193。

## 内容结构

1. 将 HST 视为交易区间，比较右肩卖出与颈线下破卖出（PDF p2179–2181）。
2. 说明颈线有多种画法，并以“市场惯性”解释反转常需第二、第三次尝试（PDF p2182–2184）。
3. 分析 HST 空头止损被触发后，从空头尝试重置为区间/原多头趋势（PDF p2185–2189）。
4. 镜像分析 HSB 右肩失守后多空共同卖出，以及第二次筑底机会（PDF p2190–2191）。

## 核心思想

- 每个 HST 都包含交易区间。传统在颈线下破处卖出，但如果右肩向下反转不够强，区间突破常失败，追空容易在低位卖出并被反转（PDF p2180–2181）。
- 更符合区间逻辑的做法是从右肩 LH MTR 向下反转时在区间顶部附近卖出；只有右肩本身形成强空头突破，颈线下破卖出的成功机会才明显提高（PDF p2180–2181）。
- HST 颈线可有三种合理画法：头部下跌低点的水平线、左肩下跌低点的水平线，以及连接这些低点的趋势线。通常彼此接近；若相距较远，先看最高一条，下破后再观察下一条是否支撑反转（PDF p2182）。
- 课程用“市场有惯性、抗拒变化”解释第一次反转常失败，需要第二或第三次尝试。可观察的对应过程是多个 HH/DT、楔形或扩张三角形顶部先后失败，直到有足够空头压力（PDF p2183）。
- 大型头肩形若持续约二十根以上，本质是成熟交易区间，原趋势恢复和趋势反转近似各 50%。多头和空头都能看到合理架构，低概率交易仍可在预期回报约为风险两倍时具有正期望（PDF p2184）。
- 头肩顶跌破颈线时，多头可能把它看成三推下跌后的楔形多头旗形或 HST 多头旗形，并买入失败下破。也就是说，空头的“触发”同时可能是多头的区间底部买点（PDF p2184）。
- 空头从右肩 LH 和颈线下破做空，常把止损放在右肩上方。若价格上破右肩，原本需要维持的 LH 失效，新的空头趋势尝试不再成立，市场应重新判断为交易区间或恢复多头趋势（PDF p2186–2187）。
- 止损触发后若上破强度普通，课程近似给出上下各 50%；若出现四根连续阳线、收在高点并强势反转此前区间下破，则多头和空头回补共同买入，较高价格概率提高到约 60%（PDF p2186–2188）。
- 空头止损并不保证一路上涨。空头还可等待右肩上方突破失败和第二次卖出架构；多头也应根据入场位置选择，若反转信号 K 线太大、买点落在区间中部，可等 HL 回调后再买（PDF p2187、p2189）。
- HSB 完全镜像：多头从右肩 HL 买入，止损在右肩下方；一旦失守 HL，新的多头趋势尝试失败，旧多头退出和新空头卖出汇合，结果转为交易区间或空头趋势（PDF p2190）。
- HSB 多头被止损后，普通下破使多空近似各 50%，强空头突破则偏空。多头可把失败的 HSB 看成三角形或潜在最终空头旗形，等待卖出高潮、失败下破或第二次 HL/MTR，再在更强多头突破后以较高概率买入（PDF p2191）。

## 重复与递进

- p2180 与 p2181 是同一 HST 颈线案例的拆分，p2181 补齐英文和中文执行逻辑，应合并读取。
- p2182 将 25B 的“颈线有多个候选、先看最近/最高目标”具体化到 HST 的三种画法。
- p2183 重复 MTR 的“第一次尝试常失败、需要多次尝试”，用市场惯性作总括解释。
- p2184 把 26A 的“二十根以上成熟区间接近 50/50”应用到 HST，并将空头触发同时解释为多头楔形旗形买点。
- p2186–2188 从止损触发后的 50/50 状态递进到强四阳线使多头约 60%，展示条件概率随突破证据更新。
- p2189 补充多头即便判断方向正确，也需避免大信号 K 线后在区间中部追入，属于位置管理而非新形态。
- p2190–2191 是 HSB 的镜像版本：HL 失效、双方卖出、状态重置、等待第二次买入。

## 可能矛盾或边界

- p2180–2181 说大多数交易区间突破失败、颈线追空大多亏损，p2184 又说二十根以上区间趋势恢复/反转约各 50%。前者可能指首次突破及其交易结果，后者指最终方向；若不区分“首次突破失败”和“最终离开区间”，会形成比例冲突。
- “每个 HST 都是交易区间”是宽泛结构定义，不表示每个头肩顶都已成熟到 50/50。形成时间短、右肩突破强时仍可有明显方向优势。
- “右肩卖出通常优于颈线下破卖出”依赖区间仍有效且止损可控。强颈线突破和跟进出现后，确认入场概率可能更高，不能一概偏好提前卖。
- 三条颈线会产生多个突破/失败突破解释。它们应当视为支撑区域而非三个独立信号，不能因连续穿线就重复叠加概率。
- “市场抗拒变化”是概括性比喻，无法独立验证。可操作内容是第一次反转失败率较高、旧趋势恢复和多次测试的观察。
- p2184 说多空架构都“好”，同时又说区间内多数架构令人失望、概率低。其成立依赖至少 2R 的回报目标和严格亏损控制，不表示多空信号都高胜率。
- 止损触发后的 50/50 不是普遍止损定律，只针对大型区间中结构失效、但新突破尚不强的状态。强突破会立即改变条件概率。
- p2188 标题仍写 50%，正文因四根强阳线改为约 60% 上涨，是有意的证据更新，不应把两数平均或任选其一。
- “多头和空头都买”“多头和空头都卖”分别由新开仓、平仓和止损构成，订单方向相同但动机不同；不能据此推断双方观点一致。
- HST 被多头视为楔形多头旗形、HSB 被多头又视为三角形/最终空头旗形，显示同一价格结构能有相反标签。最终执行仍应由位置、突破强度和跟进决定。
- p2191 的“更高概率买在强突破后”与等待失败下破/第二次 HL 是不同确认路径；强突破后止损更远、剩余回报更小，文本本页未重述这个代价。

## 翻译或抽取问题

- PDF p2180 中文只保留标题和“每个 HST 是区间”，具体颈线卖出逻辑全部要结合 p2181 才完整。
- PDF p2181 的 `BLSHS` 指 Buy Low, Sell High, Scalp，中文译成“低位买入高位卖出”但漏掉短线含义；此处实际强调在区间顶部卖。
- PDF p2182 三条颈线和先后测试位置依赖图形，纯文本只能恢复画法，不能确定实际价差。
- PDF p2183 中文加入楔形、扩张三角形顶部和再次卖出等图形标签，英文正文只明确市场惯性和多次尝试；标签位置需视觉材料。
- PDF p2184 页尾网站文字被 OCR 破坏；H2、DB、失败下破、三推和 HST 多头旗形的对应 K 线无法从文本定位。
- PDF p2186–2187 是同一 HST 止损案例拆分，p2187 补齐双方在右肩上方的后续计划。
- PDF p2188 中文夹入英文 `buy`，其意思是多头新买与空头回补共同形成买盘。
- PDF p2189 的“大信号 K 线、区间中部、HL 回调”依赖图形位置，无法从文本恢复具体入场价。
- PDF p2191 中文“多头寻找在反转上升出做多”为排版错误，应为“反转上升处”；强突破后买入与第二次尝试的标签次序依赖图示。
- PDF p2178–2179、p2185、p2192–2193 为封面、目录、分节或结束页，不提供新增独立论据。
