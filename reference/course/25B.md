# 25B：支撑阻力复测、颈线确认与双顶底失败

来源：`SRC-COURSE-01-36/25B.txt`，对应 PDF 物理页 2101–2117。

## 内容结构

1. 将反复双底/双顶解释为对支撑和阻力的确认测试（PDF p2102–2104）。
2. 用颈线突破区分原趋势、交易区间和新趋势，并给出测量移动目标（PDF p2105–2107）。
3. 说明已确认双顶底反转再失败时，可向相反方向投射测量移动（PDF p2108–2110）。
4. 比较双底成功、双底失败和上下双向测量移动（PDF p2111–2113）。
5. 用复杂交易区间说明连续双顶底、失败突破、楔形旗形和多次反转（PDF p2114–2115）。

## 核心思想

- 双顶底是支撑阻力的复测。价格曾在某一区域向上反转后，回落再测并再次向上，会加强该区域作为支撑的判断；阻力测试完全镜像（PDF p2103–2104）。
- 同一区域可以出现多个双底或双顶。课程的订单解释是多头在每次抛售时买入、空头认为下破概率低而获利回补；顶部则为空头卖出、多头减少买入（PDF p2103–2104）。
- 第二次测试只有在出现足够反向压力时才提高概率。双顶案例中，第二高点向下反转的卖压使多头放弃，课程把高概率结果的最低持续结构设为 TBTL，即约十根 K 线、两段走势（PDF p2104）。
- 双顶颈线是两次顶部之间的回调低点。若价格跌破颈线，结构形成更低低点而非多头趋势所需的更高低点，原多头趋势可能结束，后续至少转为交易区间，也可能成为空头趋势（PDF p2105）。
- 双底颈线是两次底部之间的反弹高点。若价格升破颈线，形成更高高点而非空头趋势所需的更低高点，原空头趋势可能结束，后续至少转为区间，也可能成为多头趋势（PDF p2106）。
- 颈线突破后，失望的旧趋势参与者退出、相反方向交易者入场，课程以双顶高度向下或双底高度向上投射测量移动作为持仓目标（PDF p2105–2106）。
- 交易区间同时会给多头和空头看似良好的理由，常包含 DT、DB、LH MTR、头肩底及多空旗形；这种均衡状态属于突破模式，方向仍未确定（PDF p2107）。
- 已确认的双顶反转若后来重新上穿其颈线并恢复上涨，可以向上投射测量移动；已确认的双底反转若重新跌破颈线并恢复下跌，可以向下投射测量移动（PDF p2109–2110）。
- 测量移动的颈线和投射起点往往有多个候选。课程建议先使用最近目标；若价格越过，再观察下一个目标处的反应（PDF p2109）。
- 成功双底中，第二低点略高、没有触发第一低点下方止损，说明买盘较强；突破颈线后可持有到向上测量移动，目标处多头获利、空头短线卖出会带来回调（PDF p2112）。
- 潜在双底是双向结构：深回调若守住并反转，可走向上测量移动；若筑底失败并强势下破，可走向下测量移动。课程以极宽说法称每次空头突破都是“没有筑底的双底”（PDF p2113）。
- 交易区间里没有任何单一双顶底是 100% 可靠。连续的 DB、DT 和失败突破可多次反转；应等突破足够清楚，并根据随后强趋势 K 线和跟进更新方向（PDF p2114–2115）。
- 复杂案例中，向上突破双顶同时越过双底颈线，但随后失败；三次上推可重新归类为楔形空头旗形。三根强阴线和约八根下跌使卖压占优，此时在空头突破后的回调卖出比继续相信原多头突破更有依据（PDF p2115）。

## 重复与递进

- p2103–2104 是支撑与阻力测试的底顶镜像，但顶部页进一步加入确认、TBTL 和多空对突破的竞争。
- p2105–2106 是双顶/双底颈线突破的严格镜像：原趋势高低点序列被破坏，先进入区间或相反趋势，再看测量移动。
- p2107 重复前面所有交易区间章节的“双向理由和突破模式”，本讲特有之处是区间可同时包含多个 DT/DB。
- p2109–2110 镜像说明反转形态本身失败后的相反测量移动，承接 24E 的“低概率方向突破可能延续”。
- p2112–2113 从成功双底单向目标递进到潜在双底的上下双向分支，强调先等待结构结果。
- p2114–2115 是同一编号案例的两阶段说明：先识别 1/3 DB 与 2/4 DT 的区间，后看第 6 点上破失败、三推和空头跟进。
- “测试—颈线突破—两段/测量移动—失败则反向投射”是本讲的完整统一流程，多个页面的形态标签可围绕该流程合并。

## 可能矛盾或边界

- “市场总在寻找确认”是拟人化表述。可观察含义是价格对旧水平的重复测试；不能由此证明市场具有统一目的或所有参与者都在确认同一水平。
- 支撑被多次测试既可显示买盘持续，也可能逐次消耗买盘并最终突破。本讲强调重复确认，但没有在 p2103 同时说明测试次数增多后的破位风险。
- p2104 的“确认提高概率”仍依赖第二次反转强度。仅再次到达旧高而没有卖压，不足以提高向下趋势概率。
- 颈线突破破坏旧趋势序列，只能说明原趋势可能结束；课程自己保留“交易区间或相反趋势”两种结果，不能把颈线一穿越就写成趋势确认。
- TBTL 与测量移动是不同维度：前者是时间/段数，后者是价格距离。高概率反转可能先达到其中一个，未必同时完成。
- p2109–2110 的“失败双顶/底反转”需要先有一次颈线确认，再反向重新穿越；若只看到价格位于颈线另一侧，不能省略前置过程。
- 多个颈线和投射起点使目标选择具有主观性。“先用最近目标”是管理约定，不证明该目标有唯一统计优势。
- “每次空头突破都是未筑底的双底”把所有深回调都重命名为候选双底，属于事后宽泛分类；它无法在突破前单独预测方向。
- p2114 的“突破清楚前不要入场”没有定义清楚的实体、收盘和跟进标准；结合 p2115，应至少观察强趋势 K 线、连续跟进和区间外接受。
- 交易区间有同样好的多空架构不表示每一刻概率精确相等，也不表示可以同时重仓双向交易；突破位置、区间边缘和信号质量仍影响条件概率。
- p2115 的失败上破后来形成高概率卖点，是基于新增的三根强阴线和八根下跌，不应倒推原上破从一开始就注定失败。

## 翻译或抽取问题

- PDF p2102 中文“失败的双重顶和底部”应统一为“失败的双顶和双底”，避免把 bottom 当普通名词而破坏术语对称。
- PDF p2104 的 `OT` 应为 `DT`；`ET` 很可能指 Expanding Triangle。HH MTR、颈线和强反转位置需图形定位。
- PDF p2105 的 `OT` 同样应为 `DT`；图中双顶同时被标为扩张三角形和 HH MTR。
- PDF p2109 写“上穿双顶颈线”时，必须结合此前已下破颈线的失败反转过程理解；单页文本没有明确重穿顺序，容易与标准颈线定义混淆。
- PDF p2110 中文多出“多头在双底后的反转上升附近做多”，可能是图形标签，英文正文只明确空头在空头通道的突破和回调卖出。
- PDF p2112 的颈线、两个低点、止损和测量目标依赖图示，纯文本不能恢复高度计算。
- PDF p2113 中文加入“每一次反转都是双顶或双底的一种”，英文抽取该页只明确 `Every bear BO is a DB that did not bottom`；前一句可能是译制补充或图层遗漏。
- PDF p2114–2115 的 1–6 编号和止损位置完全依赖图形；当前只能恢复相对关系，不能重建每个转折价格。
- PDF p2115 的 `OT` 两次均应为 `DT`；中文“5 跌破 3、未跌破 1”与 HL 标签顺序混乱，需图示才能判断更高低点相对哪个基准。
- PDF p2101–2102、p2108、p2111、p2116–2117 为封面、目录、分节或结束页，不提供新增独立论据。
