# 19B：支撑阻力类型、外汇报价差异与线条反转

来源：`SRC-COURSE-01-36/19B.txt`，对应 PDF 物理页 1642–1661。

## 内容结构

1. 列举移动平均线、趋势线、通道线、旧高低点、测量目标和整数位等支撑阻力来源（PDF p1644）。
2. 解释场外外汇市场不同银行、经纪商和制图方法为何使精确价格略有差异（PDF p1646–1649）。
3. 说明不同图表只是同一买卖力量的不同表示，并提出算法参与会提高目标精度的课程判断（PDF p1651–1652）。
4. 讨论趋势线的最佳拟合、重绘、平行线和信号 K 线确认（PDF p1653–1657）。
5. 以旧高低点的双顶/双底和突破回测说明水平支撑阻力（PDF p1658–1659）。

## 核心思想

- 没有一种支撑阻力始终最重要。移动平均线、趋势线、趋势通道线、旧高低点、测量移动投射和整数价位都可能成为当前主导水平，实际权重取决于上下文、可见度和该水平此前表现出的强度（PDF p1644）。
- 外汇现货交易分散在全球银行和交易场所，没有单一集中成交记录。不同经纪商接收和处理的报价非常接近却不完全相同，因此同一摆动高低点、K 线和支撑阻力可能相差少量 pip（PDF p1646–1648）。
- 不同经纪商还可能使用不同制图价格，例如买卖价中点或仅买价。这会进一步改变 K 线高低点，说明外汇支撑阻力更适合视为小范围区域，而不是所有平台共有的唯一精确价位（PDF p1649）。
- 时间图、tick 图、区间图、Renko 图和线图都在重组同一市场买卖力量。交易者不必使用相同图表才能看到有效结构；外汇报价本身存在细微差异，也降低了对单一精确图表和精确价格的依赖（PDF p1651）。
- 越明显、被越多参与者观察的线通常越重要。课程把这种共同可见性与算法使用联系起来，认为模糊线条因较少程序采用而影响较小（PDF p1652–1653）。
- 宽幅通道中的趋势线很少完美，应随着新价格出现寻找能覆盖整体趋势、获得较多触点的最佳拟合线。线条是动态工作假设，而不是必须穿过每个极值的固定几何对象（PDF p1653）。
- 在线条附近交易反转时，强信号 K 线能提高可信度。空头在趋势线阻力处向下反转可做空，止损参考最近空头腿顶部；多头在趋势线支撑处向上反转可做多，止损参考最近强多头腿底部或主要更高低点（PDF p1654–1656）。
- 若价格明显在原趋势线前反转，可能存在由对侧通道边界平移得到的平行线或另一条更相关参考线。构建通道时应同时考虑趋势线和通道线的平行关系（PDF p1655–1656）。
- 楔形通常不会呈现理想教科书外观，但构成其推进的线仍可充当支撑阻力。只有背景良好、线附近出现反转时才考虑入场；紧密空头通道中的楔形底做多仍只是较弱逆势机会（PDF p1657）。
- 旧高点和低点是最直接的水平参照：旧高附近向下反转形成双顶，旧低附近向上反转形成双底（PDF p1658）。
- 成功突破后，旧高低点或突破点常被回测。回调在突破点恢复原方向形成突破测试；分批建仓交易者在回到盈亏平衡或小利润位置退出，也会贡献该处的反向订单（PDF p1659）。

## 重复与递进

- p1646–1649 连续从市场结构、经纪商报价差异到具体制图方法解释外汇水平不完全一致，属于机制递进；不同报价例子可压缩。
- p1651–1652 从“不同图表仍表达同一力量”转到“算法参与可能使共同目标更精确”，两者共同说明精确度来自参与者共享数据和参照的程度。
- p1653–1656 围绕趋势线选择反复展开：最佳拟合、多触点、强信号 K 线、对侧平行线和止损位置分别是不同层次，不宜只保留“可重绘”。
- p1657 把趋势线思路延伸到外观不完美的楔形，新增了上下文限制和紧密通道逆势低概率边界。
- p1658–1659 从静态旧高低点反转递进到突破后角色转换和回测，并加入分批交易者退出机制。
- 本讲承接 19A 的“任一水平可能主导”和 Emini 精确目标，新增了外汇分散报价造成的精度差异，以及趋势线选择和水平回测的实际方法。

## 可能矛盾或边界

- 19A 说支撑阻力通常很精确，19B 又说外汇支撑阻力从不完全精确。两者可由数据结构解释：集中、统一报价的合约可能围绕共同目标更精确，分散外汇报价应按区域处理；但即使 Emini 也不是所有水平都能精确到一 tick。
- “任何类型都可能最重要”不表示应同时绘制所有可能水平。线条过多会产生事后解释，应优先保留当前价格附近、明显且有多次验证或清楚投射依据的候选。
- “越不明显越不重要”是共同关注度模型，不排除不明显水平恰好与大额订单重合。可见度提高概率，但不是重要性的唯一来源。
- 随进展重绘趋势线有助于适应结构，却也容易形成事后拟合。重绘应依据新增触点和整体趋势，而不是在反转后寻找任意一条恰好穿过反转点的线。
- p1655 建议未触及原线时寻找其他线，不能作为无限尝试不同画法的许可；若候选线在反转前不可合理识别，其解释价值有限。
- 强信号 K 线提高反转概率，但市场状态仍优先。紧密通道中即使有漂亮逆势信号，也可能只形成次要反转。
- 外汇不同报价相差一至数 pip 是课程案例，不是所有经纪商、货币对、时期和波动环境的固定范围。
- “计算机将很快主导外汇并使水平更精确”是课程制作时的未来预测，具有明显时间依赖。该主张未作外部核验，不能把它当作当前已证实状态。
- 突破点回测常见但非必需。强突破可能不给回调；回测发生时也可能穿回旧区间而变成失败突破。
- 分批交易者退出可以解释突破点订单的一部分，但不能排除新进场、止损和其他参与者同时作用。

## 翻译或抽取问题

- PDF p1645、p1646 的 `5/R` 是 S/R 的 OCR 错误。
- PDF p1646 的中文增加“各银行间相互套利”，英文可提取正文没有该句；即使概念合理，也不能作为该页已稳定抽取的原文论据。
- PDF p1647 的“Last trade 无关紧要”是针对无统一集中最后成交价的外汇报价结构，不应推广到集中交易市场。
- PDF p1649 的 Interactive Brokers 和 TradeStation 数据/制图方式属于课程当时示例，平台实现具有时间敏感性，未核对当前版本。
- PDF p1651 英文是 tick charts，中文译成“成交量图表”不准确；tick 图按报价或成交更新数构造，不等同于固定成交量柱。中文“每根柱含 200 个 K 线”也应为“200 ticks”。
- PDF p1652 的“计算机即将主导”带有课程年代语境；若作为当前事实引用，必须同时给出材料时间和当前市场结构证据。
- PDF p1653 中文标题和句序混杂，英文主线是：不明显的线重要性较低；宽幅通道中按最佳拟合和多触点重绘；等待突破或连续强阴线确认可提高做空概率。
- PDF p1654 的 `Stop above top of most recent bear leg` 指结构止损位置，不是固定距离。
- PDF p1655 的多个长影线和提前反转来自图例；纯文本无法确认具体哪条候选平行线在当时已画出。
- PDF p1658 的 `OT` 是 DT（double top）的 OCR 错误。
- PDF p1659 中文图例顺序严重错位。可稳定恢复的是多空镜像的突破点回测，以及分批交易者在回测处退出对反转的贡献。
- PDF p1642–1643、p1645、p1650、p1660–1661 为封面、目录、分节或结束页，不提供独立论据。
