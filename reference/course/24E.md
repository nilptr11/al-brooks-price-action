# 24E：楔形失败、少数方向加速与突破后的条件概率更新

来源：`SRC-COURSE-01-36/24E.txt`，对应 PDF 物理页 2063–2076。

## 内容结构

1. 说明楔形走向原先较低概率方向时，可能出现加速突破、两段走势和测量移动（PDF p2064–2068）。
2. 以楔形顶向上突破为例，把原先 75/25 先验更新为突破后的 50/50 成败判断（PDF p2069–2071）。
3. 镜像说明楔形底向下突破后的中性状态、二次筑底和再推一次（PDF p2072–2073）。
4. 展示部分交易者使用宽止损和分批加仓来交易失败突破（PDF p2071、p2074）。

## 核心思想

- 楔形若没有按较高概率方向反转，而是向原趋势/通道方向突破，课程预期少数路径常具有意外性，可能快速加速，形成两段走势并走向测量移动（PDF p2065–2066）。
- 下降楔形/楔形多头旗形原本更可能向上突破；若反而强势向下突破，空头趋势可能恢复。上升楔形/楔形空头旗形原本更可能向下突破；若反而大幅向上突破，多头趋势可能恢复（PDF p2065–2066）。
- 少数方向并不自动意味着大趋势。若形态小、突破小、周围仍有强反向 K 线，结果更可能只是次要突破和交易区间，多数交易者只能按短线处理（PDF p2067）。
- 交易区间里多个小突破和三次下推可构成宽泛楔形多头旗形；若卖压强、已出现 LH/LL，向下突破后可预期两段横向至下和向下测量移动（PDF p2068）。
- 形态良好且有卖压的楔形顶，事前约 75% 向下突破并形成两段横向至下；向上突破属于约 25% 的意外分支（PDF p2069–2070，承接 24D）。
- 一旦楔形顶已经向上突破，不能继续把 75% 空头先验原样使用。课程把市场重新视为多空更平衡：约 50% 突破成功并走两段、达到向上测量移动；约 50% 突破失败并向下反转（PDF p2069–2070）。
- 这个向上突破也可成为第二次顶部尝试。空头等待其失败后卖出，目标是两段下跌和向下测量移动；某些楔形因此需要第二次入场，第一次顶部反转并不完成交易（PDF p2069–2070）。
- 楔形底完全镜像：良好形态和买压事前约 75% 向上突破；一旦意外向下突破，课程把成功延续两段至向下测量移动和失败反转向上的机会各估为约 50%（PDF p2072–2073）。
- 楔形底下破可以是第二次筑底尝试，甚至还需再向下推一次才反转。多头应等待失败突破后买入，而不是因为原先 75% 向上先验就在下破过程中无条件接多（PDF p2073）。
- 部分逆势交易者不会在第一次穿越楔形边界时退出，而会预留宽止损并分批加仓，押注意外突破有约 50% 失败；底部示例把止损放在向下测量移动目标稍下方（PDF p2071、p2074）。

## 重复与递进

- p2065 与 p2066 是楔形多头/空头旗形走低概率方向后的镜像：快速突破、两段和测量移动，原则相同。
- p2067 给低概率突破加上强度边界：小形态、小突破不一定加速，可能只进入区间。这是对前两页必要的限制。
- p2068 重复 24B 的“三次尝试可按楔形功能处理”，新增 LH/LL 和强卖压使失败多头旗形进入早期空头趋势。
- p2069 与 p2070 是同一楔形顶案例的拆分文本；必须合并才能同时看到事前 75% 和突破发生后的 50/50。
- p2072–2074 镜像重复楔形底版本，并加入“可能还需一推”和向下测量移动外的宽止损位置。
- 本讲最重要的递进不是新增另一个固定概率，而是条件概率更新：突破前 75/25，观察到少数方向突破后重新评估为约 50/50。
- 两段和测量移动目标在五页中反复出现，属于同一个“意外突破可能加速”的结果模型，可合并叙述而无需逐方向重复。

## 可能矛盾或边界

- 24D 说楔形顶约 75% 向下、楔形底约 75% 向上；本讲标题又说楔形顶上破/底下破有 50% 失败。两者不是矛盾：75/25 是突破方向发生前的先验，50/50 是已经观察到低概率方向突破后的条件判断。
- 不应把两阶段概率相加或互相替换。以楔形顶为例，事前只有约 25% 出现上破；上破之后才在这部分样本中讨论约一半成功、一半失败。
- “每当市场走较低概率路径，就预期加速、两段和测量移动”措辞过强。p2067 自己说明小突破更可能区间；真正的加速分支仍需大趋势 K 线、跟进和市场状态转换。
- p2065–2066 在测量移动附近又预期获利和反向交易，说明测量移动是第一目标/磁吸位，不是突破趋势会无限延续。
- “多空力量平衡，所以 50/50”是教学近似，不是由等量订单直接测得。观察到的突破强度、跟进、所处区间和更高周期仍可使实际条件概率偏离。
- 有些楔形需第二次入场或再一推，说明第一次失败突破不是保证。等待实际的第二信号比预先假设一定形成二次顶部/底部更稳妥。
- 宽止损加仓案例只描述部分交易者行为，没有给出最大仓位、总账户风险、加仓间隔和失效条件。若文档照搬而不写这些约束，会把“50% 可能失败”误用为无限逆势摊平的理由。
- 把止损放到向下测量移动下方意味着允许低概率突破完整走到目标后才认错，单位价格风险显著增大；必须相应缩小初始和加仓仓位，课程本页没有展开。
- p2071 的空头在楔形顶上破后继续加仓、p2074 的多头在楔形底下破后继续加仓，均与“强意外突破常加速”存在风险张力。若出现强跟进和 Always In 转换，不能只依赖 50% 失败先验。
- p2068 中三次下推既被称为楔形多头旗形，又已有 LH/LL 早期空头趋势；名称指过去结构，当前市场状态可能已经转空，不能因为含“多头旗形”就继续假定多头方向。

## 翻译或抽取问题

- PDF p2065 的“Bear Wedge”在语境中是下降楔形/楔形多头旗形，因此预期多头突破；中文“空头楔形”容易误解成空头方向信号。
- PDF p2065 中文“空头获利了结，多头即将做多”是接近向下测量移动目标的图注，不能误置为刚下破时的立即反转信号。
- PDF p2066 的 `Bull Wedge` 指上升楔形，原先更可能空头突破；中文“多头楔形”同样需要说明几何方向不等于交易方向。
- PDF p2067 的推动编号、阴线与小突破位置依赖图示，纯文本只能确认“小突破和区间更可能”的限制。
- PDF p2068 标题 `Small BO` 与图中多个 Small BO 重复，正文结构较破碎；可以恢复三次下推、LH/LL、空头突破和目标，不能确定每个小突破的边界。
- PDF p2069–2070 是同一页内容拆分，p2069 中文缺少关键概率，p2070 才提供完整 75% 与条件 50/50 说明。
- PDF p2072 的 `OT` 很可能是 `DT` 的 OCR 错误，即一段上涨和双顶后空头趋势恢复。
- PDF p2071、p2074 的加仓点、宽止损距离与测量移动目标依赖图形，不能由文本重建仓位计划。
- PDF p2063–2064、p2075–2076 为封面、章节或结束页，不提供新增独立论据。
