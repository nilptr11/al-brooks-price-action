# 10A：Buying/Selling Pressure 与广义 Gap

来源：`阿布价格行为学_01-36.pdf` p600-625

## 内容结构

- p600-616：买压、卖压、可观察迹象和趋势控制力。
- p617-623：广义 gap、negative gap、测量移动及买卖压力。
- p624-625：回顾与结束。

## 核心思想

- 有利于多头的价格行为称 buying pressure，有利于空头的称 selling pressure
  （p602）。它们是聚合价格行为的可观察足迹，不等于识别具体交易者。
- 买卖会推动价格并留下压力迹象；同向压力不断累积，会提高形成相应方向趋势的机会，
  但不是确定结果（p603）。
- 趋势中一方明显掌控：牛趋势预期更高价格，熊趋势预期更低价格。课件的 `owned by`
  是控制力比喻，不是说某根 K 线归某类参与者所有（p604、p609、p615-616）。
- 买压迹象包括：bull bars 多且连续、bull bodies 相对更大、收近高点或至少中点之上、
  bull micro channel 几乎无回调、下影线，以及带 gap 和 follow-through 的 bull
  breakout（p605-607）。
- 买压增强还表现为连续 bull bars 增多、bull bodies 逐渐变大，同时连续 bear bars
  减少、bear bodies 变小（p605、p608）。相对变化比孤立一根 K 线更重要。
- 卖压是严格镜像：bear bars 更多且连续、实体更大、收近低点、bear micro channel、
  上影线、带 gap 和跟进的 bear breakout；卖压增强时 bear bodies 变大而 bull
  bodies 变弱（p610-613）。
- 多空争夺 breakout 时，可比较两侧压力以预判方向并提前入场（p614）。提前入场的
  确认较少、概率较低，必须与止损和仓位计划配套。
- Brooks 对 gap 的定义很宽：当前价格与任一支撑/阻力之间的空间都可视为 gap，远
  不限于相邻两根 K 线之间的传统跳空（p618、p621）。
- breakout point 与 pullback 轻微重叠可称 negative gap。它仍可能显示强度，但可靠
  性弱于保持开放的正 gap；gap 还可用于提出 measured move 假设（p619、p622）。
- 向上 gap 是 buying pressure，通常支持更高价格。课程把当前收盘与任意前高、或
  当前低点与更早高点之间的空间都纳入 gap；若很快测试或关闭，说明多头较弱
  （p620）。
- 向下 gap 是 selling pressure，通常支持更低价格；若立即关闭，说明空头较弱
  （p623）。

## 重复与递进

- p601、p617、p624 是总入口、gaps 分段入口和回顾。
- p602 给买卖压力定义，p603 解释形成趋势的机制，p604-616 再逐步展开迹象、动态
  变化、突破预判和趋势控制。
- p605 是完整 buying-pressure 清单，p606-608 分组重复并配例，属于总表—拆解。
- p609-613 是 p604-608 的 bear mirror，两侧应放在同一对照节而不是复制两套长文。
- p615 从单根趋势 K 线控制力推进到 p616 的整个趋势控制力，是尺度递进。
- p618-620 是 up-gap 方向的广义定义、negative gap 和不显眼 gap；p621-623 是向下
  镜像。
- p619 与 p622、p620 与 p623 分别是牛熊对称页，概念只需定义一次。

## 可能矛盾或边界

- 单根 bull/bear bar 只是压力证据的一项，不能据此断定市场已进入趋势；课程使用的
  是 bars、实体、收盘、通道、gap 和 follow-through 的证据簇。
- p614 允许预判并提前入场，不会推翻等待 breakout 确认的原则；它只是概率更低、
  价格可能更好的一种取舍。
- 下影线可表示下方 buying pressure，但一条长下影也可能是失败突破或 TR 特征；
  必须结合后续和整体 context。
- Brooks 的广义 gap 与传统 opening/inter-bar gap 不是同一范围。文档必须先声明
  使用哪一种定义，避免术语冲突。
- negative gap 字面上没有真正空白，而是 breakout point 与 pullback 的轻微重叠；
  它是 Brooks 的强度语言，不应解释成普通“向下跳空”。
- gap 支持 measured move 只是目标假设；价格可能关闭 gap 并转为交易区间或反转。
- p620/p623 的“更高/更低价格 likely”依赖 gap 是否保持、突破是否跟进，不能只因
  找到任意支撑阻力间空间就下结论。

## 翻译或抽取问题

- p600、p625：`10A` 被抽成 `lOA`/`l OA`，数字 1 与字母 l 混淆。
- p604、p609：`Clarity` 被译成“确定性”，比原文“控制方向清楚”更强。
- p605：中文“至少收盘再中点之上”有错字，应为“在中点之上”。
- p614：中文额外出现“交易区间”孤立标签，不能作为英文 bullet 的一部分。
- p618、p621：中文分别加“传统的向上/向下缺口”标题，但英文给的是 Brooks 的广义
  gap 定义，容易把两种范围混为一谈。
- p619：译者把 negative gap 解释成“出现向下缺口但市场上涨”，错误；英文指突破
  点与回调轻微重叠。
- p622：向下镜像页又把 negative gap 解释成“反向缺口但市场下跌”，同样偏离定义。
- p620、p623：多栏文字抽取打乱两种 gap 测量端点和“立即关闭”的对应关系。
