# 20A：等距走势测量、目标选择与段落边界

来源：`SRC-COURSE-01-36/20A.txt`，对应 PDF 物理页 1723–1745。

## 内容结构

1. 定义测量移动，并列出前一走势腿、交易区间高度和突破高度三类投射基准（PDF p1725–1726）。
2. 对比强趋势与深回调/区间环境中 Leg 1 = Leg 2 目标的不同含义（PDF p1727–1733）。
3. 说明测量起点常有多个合理选择，应预先计算候选并观察市场反应（PDF p1734–1738）。
4. 讨论走势腿通常从突破开始，以及交易区间如何分隔第一、第二腿（PDF p1739–1741）。
5. 以目标附近反转、目标失效和不同算法选择说明第二腿终点的事后确认（PDF p1742–1743）。

## 核心思想

- 测量移动是把先前可观察距离投射到突破方向，常用基准包括前一走势腿高度、交易区间高度和先前突破高度。Leg 1 = Leg 2 是其中最直观的等距投射（PDF p1725）。
- 当第一腿足够强，使参与者普遍预期回调后还有第二腿时，第二腿常以第一腿高度作为初始目标。它是订单集中和获利预期，不是几何必然（PDF p1726）。
- 在很强的突破趋势中，等距目标往往只是最低目标，第一处测量阻力未必能终止行情；反转尝试常失败并成为顺势旗形，市场可能继续寻找更高目标（PDF p1727、p1730–1732）。
- 当趋势较弱、位于交易区间内或回调较深约 50% 时，第二腿更可能是反弹的末端。Leg 1 = Leg 2 完成后，市场更容易转入交易区间，第三腿出现的概率相对降低（PDF p1727–1728）。
- 急速与通道结构中，第一腿可由突破构成，第二腿以通道形式展开。由于通道本身较弱且通常演化成区间，等距目标附近应更重视获利和反转风险（PDF p1728）。
- 同一图表可同时存在多个嵌套测量：两根突破 K 线实体高度、回调小区间高度、局部两腿和更大两腿都可能给出不同目标。市场参与者会在这些目标附近获利，少数也会短线反向（PDF p1728–1729）。
- 多个不同类型阻力在同一时间和价格重合时，趋势终止概率提高。即使强趋势中大多数单一阻力失效，也可在楔形、两个测量目标等形成汇合时部分或最终获利（PDF p1733）。
- 测量移动的起点和终点经常不清楚。不同算法可能采用突破起点、整个趋势腿低点、紧密通道起点或其他显著转折，因此在行情结束前未必知道市场最终采用哪个基准（PDF p1734）。
- 实务方法是列出少数合理候选目标，并观察价格是否在其中某处停顿、获利或反转。市场对目标的反应提供证据，说明哪个投射正在被更多参与者采用（PDF p1735–1736）。
- 走势腿通常从清楚突破开始，不一定从此前反向腿的绝对高低点开始。多头测量起点可高于空头腿最低点，空头测量起点可低于多头腿最高点（PDF p1734、p1739）。
- 如果第一腿后出现清楚交易区间，可把第一腿终点放在区间开始，把第二腿起点放在区间结束。区间内部的同向小腿只是区间摆动，不应自动算成整个趋势的第二腿（PDF p1740–1741）。
- 第二腿在测量目标附近出现强反转，可作为该腿结束的确认；若价格强势越过目标，则应检查更大尺度的合理腿和下一目标（PDF p1742）。
- 一个初步“第二腿”后来也可能被重新归类为交易区间内的一段，区间突破后再出现新的第二腿。段落编号是对演化结构的工作描述，而不是实时永久标签（PDF p1743）。

## 重复与递进

- p1725–1726 从测量来源定义到等距第二腿预期，属于基础递进。
- p1727–1728 都讨论深回调后的等距两腿；p1728 增加急速—通道结构、两个局部测量基准和第三腿较低概率。
- p1730–1732 连续说明强突破穿过第一阻力、寻找更高目标、逆势反转成为旗形，是同一强趋势分支的递进。
- p1734–1738 围绕起点多义性反复展开：先承认未知，再计算候选、观察反应，最后在旧目标失效时切换到预先存在的更大尺度候选。
- p1736–1737 是同一复杂图例的中英文拆分页，后页补齐多个编号起点含义，可合并。
- p1739–1741 分别说明突破起点优先、终点可能延后确认、交易区间分隔两腿，形成一套段落边界规则。
- p1742–1743 都讲第二腿终点和重新分类；前者以目标反转确认，后者保留区间后再走第二腿的分支。
- 本讲复现 19A–19C 的测量目标和支撑阻力，新增的是起终点选择、强弱环境对目标终止能力的影响，以及段落编号会随结构演化重分类。

## 可能矛盾或边界

- “等距是最低目标”只适用于第一腿足够强、第二腿预期明确的背景；弱势、区间或深回调环境中，同一目标反而可能是行情终点。
- 深回调约 50% 后仍预期等距第二腿，并不表示从原第一腿终点直接量取。第二腿从回调结束处开始，目标按第二腿自身预计高度投射。
- 第一测量目标被穿越后寻找更高目标，容易沦为不断移动目标。合理做法是事前列出由清楚结构产生的有限候选，而不是每次失效后随意重新画腿。
- 价格在某候选目标反转，只能说明该水平与订单反应一致，不能证明“多数电脑”确实采用了某个算法；这是课程的行为推断。
- 起点可选突破底部、趋势腿底部或通道起点，但候选过多会产生事后拟合。明显突破、连续强 K 线和区间边界应获得更高优先级。
- “所有趋势最终在阻力结束”若将实际终点事后定义为阻力会失去预测力。可执行信息来自多个预先可见目标汇合，而不是终点后的命名。
- 测量目标附近反向短线属于逆势交易，在强趋势中风险较高；本讲更稳健的用途是顺势获利或等待反转确认。
- 未达到测量目标不必然证明当前走势“不是第二腿”，也可能是第二腿失败。课程页给出的是重分类倾向，需要结合强反向走势和交易区间形成。
- 交易区间开始和结束也可能事后才清楚，因此“第一腿止于区间开始、第二腿始于区间结束”是整理框架，不总能实时精确标注。

## 翻译或抽取问题

- PDF p1725 中文词序重复，“前一腿高度、区间高度、前次突破高度”是三个并列测量基准。
- PDF p1727 中文段落顺序混乱。英文明确区分强突破后预期更多趋势，与较弱/区间/深回调时第二腿更可能终止反弹。
- PDF p1728 的两个测量目标分别基于两根突破 K 线实体高度和回调小区间高度，图形起点无法由纯文本精确恢复。
- PDF p1729 中文标题加入“很少完全精确”，英文可提取正文只说目标附近获利和形态嵌套；精度判断可能来自旁白或后加翻译。
- PDF p1731–1736 多次把算法选择表述为“电脑使用某目标”，当前文本没有订单或算法数据，只能保留为课程推断。
- PDF p1735 的编号 1–9 严重依赖图表；文本可恢复选择逻辑，不能在纯文本笔记中重建精确坐标。
- PDF p1736–1737 是同一页内容拆分，中文页补充了英文页未完整显示的句子，不应统计为独立论据。
- PDF p1739 的 `OT` 很可能是图中双顶（DT）标签的 OCR 错误，与核心测量规则无关。
- PDF p1740 中文末句应为“若还有第二腿，很可能从交易区间结束处开始”，不是“在交易区间结束时开始的末端”。
- PDF p1742 的“电脑判定第二腿结束”应理解为市场在目标附近强反转使该划分获得支持，而非能观察到算法内部结论。
- PDF p1723–1724、p1744–1745 为封面、目录或结束页，不提供独立论据。
