# 18B：交易区间策略、突破模式与第二段陷阱

来源：`SRC-COURSE-01-36/18B.txt`，对应 PDF 物理页 1472–1495。

## 内容结构

1. 说明交易区间为何总能同时形成多头、空头趋势论证及买卖架构（PDF p1474–1476）。
2. 给出普通交易区间的低买、高卖、短线获利和分批建仓思路（PDF p1477–1481）。
3. 定义突破模式（Breakout Mode，BOM），并将其与普通区间中约 80% 的突破尝试失败区分（PDF p1483–1484）。
4. 定义第二段陷阱：看似强劲、理应继续的第二段最终失败，并以多空陷阱和真空测试展开（PDF p1486–1491）。
5. 说明第二段陷阱在更高低点或更低高点反转后，可能成为宽幅通道的起点（PDF p1492–1493）。

## 核心思想

- 每个交易区间内部都同时包含局部多头、空头走势以及合理的买卖架构。多头趋势中的区间既可以是双底多头旗形、支持原趋势恢复，也可以构成更低高点的主要趋势反转、支持转为空头；空头趋势中的区间则镜像为双顶空头旗形与更高低点反转（PDF p1474–1475）。
- 交易区间会经历许多突破尝试，多数失败，但最终会有一次突破成功并形成趋势。中性区间最终向上、向下突破的先验概率可近似看作各 50%（PDF p1476）。
- 普通交易区间中约 80% 的突破尝试失败，因此基础执行是低买、高卖、短线获利，而不是在区间顶部追买、底部追空并期待波段。区间内多数腿很快反转，未及时获利的浮盈容易变回亏损（PDF p1477）。
- 有经验的区间交易者会根据大致顶部和底部，在下半区逐步建立多仓并退出空仓，在上半区逐步建立空仓并退出多仓；他们常用限价单、较宽止损和分批建仓押注突破失败（PDF p1478–1481）。
- 突破模式是区间压缩到足够紧、预计未来五至十根 K 线内很可能出现突破的状态。此时不能继续机械使用“80% 突破尝试失败”：突破达到利润目标与在达到目标前反转的概率约各 50%（PDF p1483–1484）。
- 突破模式中的更高低点或更低高点可以提供轻微方向背景，但并不消除突破成功和失败各约一半的不确定性；方向偏向与突破后能否达到目标是两个不同问题（PDF p1483–1484）。
- 第二段陷阱发生在市场先给出足够强的第一段，使交易者合理预期至少还有第二段；第二段随后也表现得像强突破，却在吸引追随者后失败反转（PDF p1486、p1491）。
- 多头通道上方的顺势突破约 75% 会在五根 K 线内失败。它有时仍强到形成一小段第二次上涨，但随后构成微型双顶、买入高潮或第二次做空信号，这一“第二段看似确认后反而失败”的路径就是第二段陷阱（PDF p1486–1488）。
- 区间背景下，一次强势第二段可能只是价格被吸向支撑或阻力的真空测试，而不是新趋势。强空头第二段会诱使空头在低位卖出，强多头第二段会诱使多头在高位买入，随后反转同时困住追随者（PDF p1489–1491）。
- “大涨后大跌”或“大跌后大涨”造成的双向强势和困惑本身更支持交易区间。即使强突破通常预示第二段，也必须保留区间使该第二段变成陷阱的分支（PDF p1489）。
- 如果向下的第二段陷阱在更高低点反转向上，多头是在尝试建立宽幅多头通道；如果向上的第二段陷阱在更低高点反转向下，空头是在尝试建立宽幅空头通道（PDF p1492–1493）。

- p1478–1481 连续四页展开同一参与者行为：因相信价格不会远离区间，交易者在上下半区反向交易、分批退出旧仓并用限价单建立新仓。p1480 没有新增可提取论述，纯文本可合并。
- p1483–1484 都定义 BOM 的 50% 成功/50% 目标前反转；后页只是把结果写得更直接，内容可合并。
- p1486–1488 使用同一多头通道上方突破案例。p1486 给出概率和路径，p1487 无新增文本，p1488 增加复杂两段上涨、微型双顶及第二次做空信号，属于案例递进。
- p1489–1491 分别从强空头腿、真空测试和多空对称陷阱解释第二段陷阱，核心重复但角度互补。整理时应合成“强第二段—追随—边界测试—失败反转”的机制。

- 本讲延续 18A 的“多空都有合理理由”和“80% 区间突破尝试失败”，新增了普通区间执行、BOM 例外，以及第二段陷阱这一具体失败路径。

## 可能矛盾或边界

- “最终向上或向下突破各 50%”描述中性区间的方向先验；“突破成功或目标前反转各 50%”描述 BOM 中对某次突破交易结果的条件概率。两个 50% 的样本空间不同，不能混为一条规则。
- “普通区间约 80% 突破尝试失败”与“BOM 约 50% 突破成功”不是矛盾。BOM 是区间已压缩并接近决断的特殊状态，课程明确把它列为 80% 规则的例外。
- “只做短线、不要波段”是普通区间阶段的默认策略，不表示永远不能持有突破。成功突破最终会出现，但应等待突破和跟进改变区间状态。
- 限价单、宽止损、分批建仓是高经验方法。幻灯片没有在本讲完整展开最大总风险、分批间距和仓位缩减，不能直接整理成无约束的加仓建议。
- 知道“大致顶部和底部”不等于边界固定明确。18A 已强调区间顶部和底部会产生困惑，外侧测试和假突破会扩展边界，因此反向交易仍需容纳误差。
- BOM 中的更高低点可能显示轻微多头趋势，但一次向上突破达到利润目标仍只有约 50% 的表述，说明局部结构偏向不足以把紧密区间变成高确定性趋势。
- 强第一段通常预示第二段，而“第二段陷阱”专门描述第二段也出现后却失败。它不是说所有第二段都会失败，也不是把第一段后的首次反转直接称为第二段陷阱。
- 第二段强到像突破仍可能只是真空测试，这一判断依赖交易区间背景和支撑阻力位置；在清楚趋势早期，同样强度的第二段更可能是真实延续。
- 更高低点反转只是宽幅多头通道的尝试，更低高点反转只是宽幅空头通道的尝试。后续仍需要新的高低点序列和跟进确认。

## 翻译或抽取问题

- PDF p1474 的图中文字顺序混杂。可稳定恢复的是：多头趋势内的区间同时含双底多头旗形和更低高点主要趋势反转；“此处为头肩顶”只是案例图形标签。
- PDF p1475 的 `HL MTR` 指更高低点主要趋势反转，中文顺序基本可用；多头、空头镜像关系比图中标签位置更可靠。
- PDF p1476 标题中文加入“顶部和底部从不明确”，英文可提取正文没有该句，可能来自课程旁白或后加翻译，本阶段不作为独立原文论据。
- PDF p1483 的百分号被 OCR 为 `80°/o`。英文只明确写“更高低点意味着多头趋势”；中文额外出现“更低高点意味着空头趋势”，虽为合理镜像推论，但不应伪装成该页已抽取英文原句。
- PDF p1484 的 `50o/o` 应为 50%。`profitable trade on the BO` 更准确是突破达到预设利润目标，而非任何微小浮盈。
- PDF p1486 的 `micro OT` 应为微型双顶（micro double top），不是普通 OCR 字母组合术语。
- PDF p1488 的“多头平仓促成第二根阴线”描述卖压来源之一，不表示仅由原多头退出即可解释全部成交。
- PDF p1490 的 `vacuum test` 指价格被吸向区间边界或支撑阻力、并非由持续趋势信念推动；“真空测试”需要保留这一行为含义。
- PDF p1492 中文把 `selloff reverses up` 误译成“反转向下”，正确是下跌在更高低点处反转向上，多头尝试建立宽幅多头通道。
- PDF p1493 的英文是上涨在更低高点处反转向下，空头尝试建立宽幅空头通道，中文方向基本正确。
- PDF p1472–1473、p1482、p1485、p1494–1495 为封面、目录、分节或结束页，不提供独立论据。
