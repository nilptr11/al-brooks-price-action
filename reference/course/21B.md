# 21B：反转入场的概率—回报权衡与杯柄结构

来源：`SRC-COURSE-01-36/21B.txt`，对应 PDF 物理页 1798–1819。

## 内容结构

1. 比较最早反转入场与后续确认入场的价格、风险和概率（PDF p1800）。
2. 说明主要反转需要突破旧通道并建立相反方向高低点序列（PDF p1801–1808）。
3. 展开趋势反转 80% 失败规则及提前入场、等待突破两种期望值结构（PDF p1810–1812）。
4. 将反转后的第一次回调重新描述为杯柄/旗形及更高低点或更低高点主要反转（PDF p1814–1817）。

## 核心思想

- 反转架构是新趋势最早的波段入场，因入场靠近极值，潜在回报最大、结构止损常较近；代价是当时证据最少、成功概率通常较低（PDF p1800）。
- 交易者不必执着最早入场。反转之后还会出现突破、回调、旗形和趋势延续等许多架构，它们可能以较差价格换取更高确认，整体同样具有交易价值（PDF p1800）。
- 多头趋势由上升通道内更高高点、更高低点构成。要真正转为空头趋势，价格需先跌破多头通道，再从一个更低高点向下反转，并逐步建立更低高点和更低低点（PDF p1801–1802）。
- 空头趋势则由下降通道内更低高点、更低低点构成；出现新信息时需更新候选趋势线。多头主要反转需要突破空头通道并建立更高低点序列（PDF p1803，结合 21A p1791 的镜像规则）。
- 趋势惯性使多头趋势中的多数阻力、空头趋势中的多数支撑失效；但趋势最终仍会在某个多重目标处结束。楔形、测量目标和旧高低等汇合比单一水平更值得关注（PDF p1804–1807）。
- 只要逆势摆动仍留在原通道内并守住主要结构点，它们就是回调。多头趋势结束至少需要跌破通道并跌破主要更高低点，之后才进入交易区间或空头趋势（PDF p1808）。
- 课程的 80% 规则称，大多数趋势反转尝试会失败。部分交易者仍反复尝试最早反转，但必须在错误时快速退出，等待某一次真正获得突破和跟进（PDF p1810）。
- 提前反转交易的经济结构是低概率、大回报。课程示例用约 40% 成功率和至少 2:1 回报风险比说明：即使多数交易失败，足够大的赢家仍可能补偿失败（PDF p1811）。
- 若希望更高概率，应等待强突破。课程给出强突破后盈利概率常约 60%，但合理止损离入场更远；确认提高的代价是更差价格和更大的每单位仓位风险（PDF p1812）。
- “没有完美交易”表示至少一个变量会较差：早入场通常概率低，确认入场通常止损远，近止损又更容易被触发。交易选择是在概率、回报和止损距离之间交换（PDF p1812）。
- 反转最终会出现第一次回调。课程把底部反转本身称为杯体、回调称为柄部；柄部同时是多头旗形，常形成更高低点主要趋势反转，向上突破柄部提供较晚但更确认的波段买点（PDF p1814–1816）。
- 顶部结构完全镜像：向下反转构成杯体，随后的向上回调构成柄部/空头旗形，在更低高点向下突破柄部可作为主要反转做空（PDF p1817）。

## 重复与递进

- p1801–1803 延续 21A 的通道突破和新高低点序列，新增反转最早入场与新通道形成的直接联系。
- p1804–1808 以多头、空头镜像重述趋势惯性、最终支撑阻力及结构失效；p1808 给出更具体的“通道突破加主要更高低点破坏”条件。
- p1810–1812 从 80% 失败递进到 40% 可选交易和 60% 强突破确认，看似重复概率，实际对应不同筛选和入场阶段，但课程未明确分母。
- p1814–1816 三页几乎重复底部杯柄定义，最后一页才增加柄部突破、旗形和 HL MTR；可合并。p1817 是顶部镜像。
- 杯柄部分本质上重复“反转—回调—第二次入场/旗形突破”的既有思想，新增价值主要是把同一路径映射到传统杯柄术语。

## 可能矛盾或边界

- “最早反转风险小”指止损距离可能较小，不代表账户风险自动较小；低成功率和频繁止损仍可能造成较大累计损失。
- 80% 反转尝试失败意味着约 20% 成功，而下一页又用 40% 成功率。最合理的解释是前者涵盖所有尝试，后者是筛选后的可交易架构，但幻灯片未给出清楚分母，必须作为待核口径保留。
- 21A 还说“下一次反转约 40% 成为主要反转”，同样可能是特定趋势恢复后的条件概率，不能直接与 80% 规则合并。
- 2:1 回报风险、40% 胜率在忽略交易成本时具有正期望，但只有实际平均赢家和输家符合假设才成立；不能把计划目标当成已实现分布。
- 等待强突破提高到约 60% 同样缺少样本口径。止损更远时可通过缩小仓位固定账户风险，因此“更高概率意味着更大风险”主要描述每单位仓位距离。
- “所有多头趋势最终在阻力结束、所有空头趋势在支撑结束”具有事后命名风险；可执行性来自反转前可见的多个目标汇合。
- 杯柄在本讲被极广义化为“任何反转加后续回调”。这种重新命名本身不增加概率，真正有用的是回调形成 HL/LH、旗形突破及跟进。
- 柄部突破仍可能失败，尤其在交易区间或原趋势惯性强时；它比最早反转更确认，但不是必然成功。

## 翻译或抽取问题

- PDF p1800 中文把 LL MTR、楔形底和最早买点标签混在一起；可确认的是最早反转低概率、大潜在回报，图中具体形态层级无法纯文本还原。
- PDF p1803 的“每有新信息就持续绘制新线”应限制为更新合理候选，不是无限增加线条。
- PDF p1808 中文“从高于前一高点的低点反转”应为从高于前一更高低点的位置反转，英文是 `above prior higher lows`。
- PDF p1810、p1811、p1812 的 80%、40%、60% 很可能对应不同交易集合，但材料未显式定义统计口径，是后续全局矛盾表的重要项目。
- PDF p1812 图中 `Lower risk, but lower probability` 与 `higher probability ... more risk` 指早晚入场的止损距离/概率交换，中文排版把标签顺序打散。
- PDF p1814 英文写底部的 cup 是 `upside down`，但标准底部杯柄通常画作 U 形，p1815–1816 又删除该括注。仅凭文本无法判断是图形方向语境、讲者特殊用法还是幻灯片错误，应保留待核。
- PDF p1815 中文增加“柄部：看涨旗形，常伴 HL MTR”，英文可提取正文在 p1816 才完整出现，属于跨页补充。
- PDF p1798–1799、p1809、p1813、p1818–1819 为封面、目录、分节或结束页，不提供独立论据。
