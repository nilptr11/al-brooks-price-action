# 22C：移动平均缺口 K 线、广义双顶底与反转卖压

来源：`SRC-COURSE-01-36/22C.txt`，对应 PDF 物理页 1903–1917。

## 内容结构

1. 定义多头趋势回调中的移动平均缺口 K 线（MAG），并把它作为趋势可能进入最后一段的预警（PDF p1904–1906）。
2. 将所有顶部、底部 MTR 归纳为宽松定义的双顶或双底测试，强调第二个极值不必与第一个完全相等（PDF p1907–1910）。
3. 比较未确认的双顶 MTR、强空头突破和 LH MTR 三个卖出层级（PDF p1911–1912）。
4. 用时间和价格两种方式定义足够卖压，并说明趋势线在 MTR 识别中的提醒作用（PDF p1913–1914）。
5. 说明 DT/HH MTR 后常出现 LH MTR，为退出者或保守交易者提供再次卖出机会（PDF p1915）。

## 核心思想

- 多头趋势中的 MAG 是一根最高价完全位于移动平均线下方的 K 线，即其最高点与均线之间仍有缺口。它表明空头曾强到把整根 K 线压到均线下方，是空头可能很快取得控制权的预警（PDF p1905）。
- MAG 后的反弹常被课程视为多头趋势的最后一段。交易者不是在 MAG 本身机械做空，而是等价格重新测试旧多头高点，并寻找 HH、DT 或 LH MTR 的向下反转信号（PDF p1905–1906）。
- 完整卖出架构需要环境和信号配合：环境包括先有跌破多头趋势线的下跌、可能测试均线、原趋势恢复后再次测试顶部；信号则是顶部附近出现有力的向下反转 K 线。均线测试不是绝对要求（PDF p1906）。
- 课程把所有 MTR 顶部宽泛地视为双顶，把所有 MTR 底部视为双底，因为市场都在再次测试旧趋势极值，并选择继续突破还是反转。第二个顶/底可以略高、相等或略低，分别对应 HH、DT、LH 或 LL、DB、HL 变体（PDF p1908–1909）。
- 形态名称不是决策核心。近似双顶略创新高时既可叫不完美双顶，也可叫 HH MTR；关键仍是它是否在合适环境中形成可执行的反转架构（PDF p1910）。
- 强反向突破会提高后续 MTR 的成功概率。示例中即使回调没有触及均线，约十一根 K 线的紧密交易区间也足以中断多头趋势并使市场趋于中性；随后三根强阴线和强卖出信号使更低价格的概率上升（PDF p1911）。
- 空头若要把多头趋势转为空头摆动，需要展示足够卖压。常见证据是回调到均线附近、下跌跌破多头趋势线，以及大阴线或持续跟进；卖压不足时更可能只有次要反转（PDF p1912）。
- 卖压可通过价格或时间表达：少数很大的空头趋势 K 线代表快速的价格位移；约十根或更多较弱空头 K 线、或长时间横盘而价格无法继续上涨，则用时间证明多头失去控制（PDF p1913）。
- 有趋势时应尽早画趋势线；趋势线被跌破后若价格再次上涨或横盘，可按新结构重画。它的用途不是直接产生交易，而是提醒交易者注意“突破旧线—旧趋势恢复—顶部测试再次失败”的 MTR 序列（PDF p1914）。
- DT 或 HH MTR 后通常还有一次 LH MTR。较早卖出失败、提前退出或选择等待确认的交易者，可以在后续更低高点获得另一卖出机会（PDF p1915）。

## 重复与递进

- p1905–1906 从 MAG 定义递进到完整 MTR 环境：MAG 是预警，顶部测试和强反转信号才是入场架构；这一边界是本讲新增重点。
- p1908–1910 重复 21D 的“所有顶部/底部都是双顶/双底变体”，本讲用 HH/DT/LH 与 LL/DB/HL 映射进一步说明第二个极值无需精确相等。
- p1908 与 p1909 是顶部和底部的镜像表达，可以合并为“旧极值测试”模型；底部额外指出强多头突破后的 HL MTR 概率较高。
- p1911 再次展示 22B 的“确认后概率更高”，新增的是十一根紧密交易区间可替代均线触碰，说明趋势暂停按功能而非单一指标判断。
- p1912–1913 重复 21C、22A 的反向压力尺度，并更明确地区分时间证据与价格证据。
- p1914 重述 22A 的三阶段 MTR：趋势线突破、原趋势恢复、旧极值测试失败；新增的是持续重画趋势线作为观察流程。
- p1915 与 22B 的重新入场逻辑相接：DT/HH 是较早尝试，LH 是常见的第二次机会。

## 可能矛盾或边界

- MAG 是警告而非卖出信号。多头趋势可能在整根 K 线落到均线下方后仍恢复；缺少顶部测试失败和强向下信号时，不能据此宣布主要反转。
- “MAG 后的反弹往往是最后一段”是经验性倾向，没有给出频率和条件；趋势很强时也可能有多个均线缺口与新高。
- p1906 同时列出均线测试又注明并非绝对要求；应把均线视为常见环境参考，而不是 MTR 定义组成部分。
- “所有 MTR 顶部都是双顶”采用极宽定义，把略高或略低的第二次测试都纳入。此说法有助于统一形态，但若没有测试对象、卖压和确认条件，分类本身没有足够预测力。
- p1909 所说强多头突破后的 HL MTR 概率更高，与 p1908 的“原则相同”并不矛盾：结构分类相同，但前置突破强度改变条件概率。
- 十一根紧密交易区间“使市场中性”是功能判断，不表示多空概率必然精确 50/50；后续强突破仍决定方向。
- 足够卖压的时间尺度“约十根或更多”不是固定门槛。少量极强 K 线可以更快建立压力，许多高度重叠、双向尾巴明显的 K 线也可能只表示中性而非空头控制。
- 画趋势线具有主观性；不断重画可适应新结构，也可能导致事后选择。其有效用途是组织观察，不应把任意一条线的轻微跌破当作反转证据。
- “DT/HH 后通常有 LH”不是保证。强突破可能不给回调，旧趋势也可能直接恢复；重新入场必须等实际信号，而非预先假定一定出现。

## 翻译或抽取问题

- PDF p1905 的 MAG 英文定义清楚：K 线最高价完全低于移动平均线。中文文字顺序混乱，但不应误读为仅收盘低于均线。
- PDF p1905–1906 的具体 MAG、HH、LH 和卖出点依赖图形位置，纯文本不能复原各标签对应的 K 线。
- PDF p1906 的 `Bears exit` 可能是图中特定位置的空头退出标注，与核心定义关系不清，不能据此总结通用退出规则。
- PDF p1908 的 `Failed BO above HST` 中 `HST` 可能是头肩顶标签，抽取不足以确认其图形边界。
- PDF p1909 的多个 HLMTR 标签在抽取文本中重复，无法区分哪个跟随强多头突破。
- PDF p1911 的 `Sell OT MTR` 很可能是 `Sell DT MTR` 的 OCR 错误。
- PDF p1912 标题被抽成 `P•neversal`，结合正文应为反转主题，不影响内容含义。
- PDF p1913 的“约 10 根或更多”中文把弱势、交易区间和空头 K 线标签混排；英文表达的是时间型卖压可以由交易区间或约十根以上较弱空头 K 线表现。
- PDF p1914 图中可能有多条趋势线，但纯文本只能确认课程要求随着结构变化重画，不能复原线的具体锚点。
- PDF p1903–1904、p1907、p1916–1917 为封面、提要或结束页，不提供独立论据。
