# 01–52 概念索引

## 一、课程思想的总骨架

课程并不是由大量彼此独立的 K 线形态组成，而是反复围绕同一条判断链展开：

1. **先判断市场状态**：强突破、紧密通道、宽幅通道、交易区间或状态转换。
2. **再解释当前位置**：相对趋势、区间边界、支撑阻力、旧高低点、测量目标和剩余空间。
3. **识别当前证据**：趋势 K 线、信号 K 线、突破跟进、回调深度、买卖压力和失败测试。
4. **构造交易者方程**：把概率、潜在回报、结构风险和成本放在一起，而不是只看形态名称。
5. **选择执行方式**：stop/limit/market 入场，scalp/swing 管理，是否等待确认或计划内分批建仓。
6. **持续更新前提**：原前提有效则依赖结构止损；前提转为区间、反向突破或 Always In 翻转时改变目标、退出或重新入场。
7. **控制个人风险**：通过实际止损、仓位、总敞口和执行纪律，防止正常统计亏损变成灾难性亏损。

这条链在 01–36 中被拆成定义和专题，在 37–47 中按市场状态重新整合，在 48–52 中通过日内案例、剥头皮和错误管理再次应用。

## 三、容易混为一谈的同名概念

### 1. 风险

- **初始风险**：入场到原始保护性止损的事前距离。
- **实际风险**：交易完成后回看，价格曾向不利方向走出的最大距离；它是事后量。
- **账户风险**：止损距离 × 仓位，并受滑点、加仓和相关敞口影响。
- **个人风险**：因恐惧、希望、规则破坏和执行错误造成的额外亏损。

课程后半反复出现的“低风险不是好交易”，通常是在批评过紧的技术止损；它不能被理解为反对低账户风险。

### 2. 突破

- 越过某条线只是最弱意义上的突破尝试。
- 大趋势 K 线、收于极值、缺少重叠和良好跟进提高突破可信度。
- 普通交易区间中的突破尝试、成熟突破模式中的决断，以及强趋势中的新突破不是同一个统计集合。
- 突破已发生后，原先关于突破方向的先验必须按跟进重新更新。

### 3. 反转

- 一至数根反向 K 线可能只是小反转或回调。
- MTR 要求旧趋势、反向压力、趋势线/结构破坏、旧极值测试和新的 HL/LH 等证据组合。
- 即使 MTR 形态完成，结果也可能只是交易区间，而非形成相反趋势。

### 4. 交易区间

- 回调、宽幅通道和交易区间是连续谱，不是互斥的静态标签。
- Tight Trading Range 是压缩状态；当它进入 Breakout Mode 后，不能继续按普通区间的每次边界失败来管理。
- 宽幅通道既有趋势偏置，又允许更多双向交易，应同时保留两层描述。

### 5. 止损

- **保护性止损**是最坏情景下的硬退出机制。
- **结构提前退出**发生在市场已在止损前证明原前提失效时。
- **追踪止损**在新趋势结构确认后保护已得优势。
- **“臭鼬止损”**是既不及时认错、也不接受原波段止损，临时放在正常噪声中的折中位置。

## 四、完整思想之间的依赖关系

- 形态依赖背景：同一个 H2、楔形或双底，在强趋势、宽通道和区间边缘具有不同含义。
- 概率依赖事件定义：40%、60%、70%、75%、80%、90% 只有在分母、目标距离和观察时点一致时才可比较。
- 管理依赖入场构造：早入、确认后入、scalp、swing、分批建仓分别对应不同止损与目标，不能事后任意互换。
- 状态判断依赖持续更新：强趋势最终会失望，区间最终会突破，但在新证据出现前仍按当前状态管理。
- 风险控制独立于方向预测：即使方向判断正确，仓位过大、止损无结构依据或过早止盈也会使交易方程变差。
- 课程中的完整理解不是背诵每个比例，而是理解这些比例如何随状态、位置、证据和目标条件改变。
