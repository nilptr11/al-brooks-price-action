# 30B：概率—回报—风险三变量、加仓交换与深回调位置

来源：`SRC-COURSE-01-36/30B.txt`，对应 PDF 物理页 2321–2343。

## 内容结构

1. 将交易计划分解为概率、回报、风险三变量，并说明早晚入场交换（PDF p2322–2324）。
2. 比较初学者偏好小风险与熟练者按期望选择高概率或高回报风险比（PDF p2325–2326）。
3. 说明波段持有和分批加仓如何改变交易者方程式（PDF p2327–2328）。
4. 用多空突破案例展示确认程度、概率、止损距离和剩余目标的动态变化（PDF p2329–2332）。
5. 以多头/空头趋势的深回调说明价格位于半程和上下三分之一区域时的 60/40 与 2R 交换（PDF p2333–2341）。

## 核心思想

- 每笔交易必须同时评估三项：成功概率、潜在回报、风险；也可压缩为“概率＋回报风险比”。任何只比较目标与止损、不估计概率的方案都不完整（PDF p2323）。
- 早期入场通常止损近、剩余目标大，回报风险比好，但概率低；后期确认入场概率高，却因止损远、目标已走掉一部分而回报风险比差（PDF p2323）。
- 课程把对手双方的选择描述为变量交换：一方接受高概率和弱回报风险比，另一方接受低概率和强回报风险比；不可能同时长期占有高概率、大回报和小风险（PDF p2324）。
- 高概率弱 RR 与低概率强 RR 都可能正期望，关键是管理和实际数学。课程认为初学者常因账户小而只选止损近的交易，却忽略这类早期架构可能有 60% 以上失败，并因管理不当亏损（PDF p2325）。
- 熟练交易者根据当时优势和持有条件选择变量。例如时间短时可选择更快实现、小目标的高概率短线；不执着抓极值，而等待突破或回调，以更远止损换取课程所称约 60% 以上概率（PDF p2326）。
- 从短线改为波段可改善平均回报：初始风险相同，短线在约 1R 退出，波段持有获得更大平均赢家；只要额外回报足以覆盖略低的止盈前成功率，期望会改善（PDF p2327）。
- 分批加仓是另一种期望结构。第二次在更有利信号处加仓可提高整体头寸最终盈利/保本的机会，但总风险增加、平均剩余回报下降；示例在反弹回到第一入场附近时，第一笔保本、第二笔盈利（PDF p2328）。
- 强突破和跟进提高向测量移动发展的概率，但结构止损仍在突破前极值之外，因此单位价格风险较大。突破前 DB/HL MTR 或 DT/LH MTR 止损近、回报大，但区间突破多失败，概率较低（PDF p2329–2330）。
- 强突破后的第一次回调是旗形，通常仍偏顺势；若回调本身成为 TTR，方向优势较刚突破时略低，但仍高于最早的区间内逆势尝试（PDF p2329–2330）。
- 课程把极高胜率与短线联系起来：若有人约 90% 盈利，通常只取很小目标，让对手获得更好的 RR；自己的多数交易小赚，对方多数交易小亏但少数可能大赚（PDF p2331）。
- 计划委托时，入场到保护止损的距离是计划风险，入场到限价目标的距离是计划回报；二者可按 tick/pip 计算。概率最重要却不确定，常被交易者忽视（PDF p2332）。
- 在深回调中必须向左看整个趋势结构。多头趋势即使短期已 Always In Short，只要价格仍高于整个多头波段前低，就仍处于更大多头通道；空头要跌破整个波段最低结构止损，才真正结束该通道（PDF p2333、p2335）。
- 多头波段回调尚在中点上方，即回撤小于约 50% 时，多头创新高概率大于 50%，但买入止损较远、剩余回报较小，RR 较弱（PDF p2334）。
- 回调越深到波段下半部、尤其强抛售底部三分之一区域，多头止损更近、到旧高回报可达约 2R，但多头创新高概率降到约 40%，空头创新低约 60%（PDF p2334）。
- 不能因深度回调价格便宜就假定有足够买盘。底部三分之一区域潜在资金数量不可知，必须等实际强向上反转使市场回到 Always In Long，才证明多头足以形成 HL 和宽幅多头通道（PDF p2336）。
- 空头趋势完全镜像：价格仍低于整个空头波段前高时，即使短期 Always In Long，也仍是更大空头通道；回调不足 50% 时空头概率高但 RR 差，升到顶部三分之一时空头创新低概率约 40%、潜在回报约 2R（PDF p2337–2341）。
- 顶部三分之一区域同样存在未知卖盘；只有实际强向下反转、恢复 Always In Short，才证明空头足以形成 LH 和宽幅空头通道（PDF p2341）。

## 重复与递进

- p2323 重复 30A 和前面所有“早入场/确认入场交换”，本讲正式归纳为概率、回报、风险三变量。
- p2324–2326 从对手变量交换递进到初学者和熟练者选择，行为叙事重复较多，数学核心仍是期望值。
- p2327–2328 给出改善期望的两种路径：延长赢家提高平均回报，以及加仓提高整体盈利概率但牺牲 RR。
- p2329–2330 是多头/空头突破镜像，三阶段均为区间内早入场、强突破确认、突破后 TTR 回调。
- p2331 把“高概率必付代价”推到 90% 短线极端，是 p2324 的例子而非新公式。
- p2333–2336 与 p2337–2341 是深回调的多空镜像，围绕整个波段边界、50% 中点、上下三分之一和反转确认。
- 40%/2R、60%/弱 RR 再次重复 MTR 和交易者方程式章节，但本讲新增把概率变化映射到波段内价格位置。

## 可能矛盾或边界

- 每笔成交确有买卖双方，但双方不必是在同一周期上交换概率和 RR。对手可能做市、对冲、跨市场套利或管理不同期限头寸；“一家要概率、另一家要 RR”是简化模型。
- “低风险交易 60% 以上亏损”只适用于课程所指早期、近止损、低概率架构，不适用于所有小账户风险或所有近止损交易。
- “等待突破或回调概率 60% 以上”同样需要强突破、跟进和正确环境；普通区间假突破或弱回调没有统一 60% 成功率。
- 分批加仓不会改变未来价格路径的概率，只会改变平均入场价、仓位和“整体头寸达到保本/盈利”的概率。若文档写成提高市场上涨/下跌概率，就是概念错误。
- 加仓增加最大仓位和累计风险。只有事先定义总止损、每次规模和最大尝试，才能谈改善期望；否则可能用更大尾部亏损换取更高小盈利率。
- 高概率不必然等于短线，90% 也不是可普遍达到的短线胜率。p2331 是变量交换的极端示意，缺少样本和成本。
- p2332 所说 RR“通常精确”只指计划委托距离。实际成交、滑点、跳空、部分退出、目标未成交和提前止损会使实际 RR 不精确。
- 计划回报并非平均回报，止损距离也不等于账户风险；账户风险还取决于仓位，交易者方程式应使用实际分布。
- 价格高于多头前低却 Always In Short、低于空头前高却 Always In Long，说明 Always In 是当前腿/较低层级状态，通道是更大结构；仓库必须标明尺度，避免看似矛盾。
- 50% 回撤是连续位置而非离散开关。概率不会在穿越中点时突然跳变，课程的 60/40 和 2R 是近似区域模型。
- p2334 同页混合“多头概率大于 50%”“约 50%”和底部三分之一 40%，对应不同深度位置；若不按图中区域拆开，会误以为自相矛盾。
- 深回调底部/顶部三分之一的 60/40 依赖原趋势、抛售/反弹强度和结构止损，不能泛化到任意价格区间。
- “总有未知资金准备买卖”无法验证，且存在资金不代表会立即成交。课程自己要求先看反转强度，后者才是可观察确认。

## 翻译或抽取问题

- PDF p2324 的“两个机构”是抽象对手模型，不一定真是恰好两家机构；中文直译容易具象化。
- PDF p2325 的 `Stop`、`6 0` 为图中止损和概率轴标签，不是正文断句。
- PDF p2328 的两个空头入场、第一笔保本和第二笔盈利依赖图中价格位置，纯文本不能计算总体最大风险。
- PDF p2329 中文“由于处于紧密交易区间（及三角形）”对应突破后回调区域，英文只写 bull flag/稍低概率，具体形态标签可能在图中。
- PDF p2330 的 `OT` 应为 `DT` 的 OCR 错误；中文标题“成功率：空头突破后更高”是图页概括。
- PDF p2332 的 `RR Usually Exact` 应限定为下单时的计划距离，中文若不加“计划”容易误导为实际结果精确。
- PDF p2333–2335 的 50% 线、底部三分之一、风险与回报箭头依赖图形，文本只能恢复区域相对关系。
- PDF p2334 英文 `As long as PB stays above 50%` 指价格保持在原多头波段中点上方，即回撤幅度小于 50%；中文标题表达回调幅度，二者参照方式相反但含义一致。
- PDF p2337 中文把 `Broad Bear Channel` 与“多头突破”混排，正确结构是强多头波段仍可能只是宽幅空头通道内回调。
- PDF p2339–2341 的顶部三分之一和下半部标签依赖空头波段图，方向需以英文为准。
- PDF p2321–2322、p2342–2343 为封面、目录或结束页，不提供新增独立论据。
