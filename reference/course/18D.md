# 18D：紧密交易区间与限价订单市场

来源：`SRC-COURSE-01-36/18D.txt`，对应 PDF 物理页 1536–1558。

## 内容结构

1. 定义长期横盘的紧密交易区间及“闲置资金”现象，并给出功能性识别方法（PDF p1538–1541）。
2. 将交易区间视为水平通道，区分宽幅区间与过紧区间可使用的订单类型（PDF p1542–1546）。
3. 定义限价订单市场（Limit Order Market，LOM）及其反向短线、宽止损、分批建仓方式（PDF p1548–1553）。
4. 讨论均线位置带来的轻微方向偏向，以及紧密区间突破前后概率与风险的变化（PDF p1554–1556）。

## 核心思想

- 紧密交易区间是水平的紧密通道，可以出现在所有市场和周期。股票在日、周、月线大幅下跌后，常会横盘二十根或更多 K 线筑底；因资金长时间缺少回报，交易者称之为“闲置资金”或“死钱”（PDF p1538–1539）。
- 紧密交易区间的实用定义与紧密通道相同：如果短线交易者在区间底部附近用停止单买入、顶部附近用停止单卖出仍难以获利，说明确认发生时价格已经太高或太低，区间可交易空间不足，因而属于紧密区间（PDF p1540）。
- 对大多数交易者，过紧区间的最佳处理是等待任一方向的突破及有效跟进。可在突破发生时入场，但确认较少、概率较低；等待强突破后的回调，通常以更充分证据换取较差价格或更宽风险（PDF p1541）。
- 交易区间本质上是水平通道，也有宽幅和紧密之分。宽幅区间边缘空间足够时，可以在顶部、底部反转信号处使用停止单，也可以用限价单押注边界突破失败；强反转已经发生时还可用市价单进入（PDF p1542–1545）。
- 如果区间过紧，停止单、限价单和市价单都可能缺少足够利润空间，大多数交易者应完全不交易并等待突破（PDF p1546）。
- 限价订单市场是紧密但仍大到足以短线获利的区间。停止单入场会买得太高、卖得太低，参与者因而在下半区 K 线下方限价买入、上半区 K 线上方限价卖出（PDF p1548）。
- LOM 中多空双方都预期反转而非立即突破，采用短线利润目标、较宽的波段级止损，并可能分批建仓。其交易逻辑依赖大行情短期内不太可能发生（PDF p1549、p1552–1553）。
- 课程再次用相对尺度判断区间是否足够大：多数 K 线本身应大于最小短线目标，区间总高度还应约为最小短线目标或平均 K 线高度的三倍量级（PDF p1550）。
- 即使某个紧密区间理论上足够进行限价短线，大多数交易者仍不应参与：利润小、操作密集，少数错误就可能抵消许多小盈利（PDF p1551）。
- 紧密区间上下突破概率接近 50%。区间大部分位于均线上方时只是弱多头趋势，向上突破概率略高；大部分位于均线下方时只是弱空头趋势，向下突破概率略高（PDF p1554）。
- 紧密区间仍可孕育主要趋势反转和大行情。突破前押注方向，止损距离可能小但成功概率也低；等待强突破后再顺势，成功概率提高，却因合理止损更远而承担更大的每单位仓位风险（PDF p1555–1556）。

## 重复与递进

- p1538–1539 从长期持有者的“闲置资金”视角说明紧密区间利润潜力小，p1540–1541 转为短线交易者的功能性定义和等待突破方案，属于视角递进。
- p1542–1546 连续对比宽幅与紧密区间：宽幅区间可用停止、限价或强反转后的市价单，过紧区间则不交易。各页订单类型不同，不应简单视为重复。
- p1548–1549 先定义 LOM，再列出短线、宽止损、分批三项行为；p1552–1553 用 EURUSD 案例重复同一组合，并增加具体目标和分批距离。原则上可合并，数字只保留为案例。
- p1550–1551 再次重复 17B 的区间高度相对最小短线和平均 K 线尺度；新增重点是“满足最低高度仍不代表适合多数交易者”。
- p1555–1556 是 Emini 与外汇的镜像案例，文字几乎相同，可合并成“确认越多，概率提高但止损更远”的一条规则。
- 本讲延续 17B 的紧密交易区间尺度和 18B–18C 的限价反向、BLSHS 行为。新增价值是将“过紧不可交易”和“紧但可供专家限价短线的 LOM”明确分层。

## 可能矛盾或边界

- 紧密交易区间不等于限价订单市场。前者包含完全没有可交易空间的区间；只有“紧密但仍足够大”的子集才被本讲称为 LOM。
- “唯一可能盈利的方式是限价单”与 p1553 的“限价或市价单入场”字面上存在张力。较合理的限制是：停止单在狭窄区间中通常确认太晚；限价单用于提前反向，市价单可能用于强反转已经显现时即时成交。但幻灯片没有完整说明市价单何时仍保有足够空间，整理时不应把两者无条件并列。
- 宽幅交易区间可用三种订单，不表示任意位置都可使用。停止单应在边缘反转信号后触发，限价单应在上下半区押注失败突破，市价单应等待强反转；区间中部仍缺少优势。
- “等待突破后交易”也有层级：突破瞬间入场较早但确认少，等待跟进或回调确认更多但止损可能更远。不存在同时获得最高概率、最好价格和最小风险的方案。
- p1550 对最低区间高度写成“最小短线三倍或平均 K 线三倍”，而 17B p1415–1416 明确写为同时满足两者的约两至三倍。该处课程文本本身口径不一致；从同页案例同时比较两个基准看，更保守的整理应采用“至少覆盖两项尺度中的较大要求”，并将确切倍数保留为经验范围。
- “均线上方略偏多、均线下方略偏空”只是微弱修正。若优势足够大，市场就不会继续保持紧密区间，因此不能把均线位置升级为高概率突破信号。
- “突破前风险小、突破后风险大”谈的是止损距离，不是总体账户风险。确认后可通过缩小仓位保持账户风险不变，本页未展开仓位计算。
- LOM 交易者认为大行情“不太可能”只是当前条件概率；紧密区间最终仍可能产生很大突破，因此宽止损和分批不能替代最大风险限制。
- “闲置资金应去别处交易”是机会成本建议，适用于缺少收益空间的长期盘整，不表示该资产基本面或未来方向已经确定。

## 翻译或抽取问题

- PDF p1538 的 `After big move on daily, usually down` 句法简略，结合后文应理解为日线大幅波动、尤其急跌后常长期横盘，而不是所有大行情之后价格通常继续下跌。
- PDF p1540、p1541 的 `stop orders` 是确认式入场停止单，不是保护性止损；中文“止损订单入场”容易混淆。
- PDF p1541 图中文字顺序混杂，可确认的是“等待突破及跟进”与“突破入场较早但概率较低、回调入场确认更多”的对比，无法仅凭文本复原每个箭头。
- PDF p1544 中文买卖词序有错位。英文明确是区间顶部在 K 线上方或弱多头旗形处卖出，区间底部在 K 线下方或弱空头旗形处买入。
- PDF p1549、p1552 的标题写成 `LCM`，正文和章节目录均为 Limit Order Market，显然应统一为 `LOM`。
- PDF p1550 英文列出 `either`，但同页案例同时验证最小短线和平均 K 线两项，且与 17B 的 `AND` 不一致。这是需要在后续全局核对表中保留的实质口径问题。
- PDF p1550 的 Emini 固定 tick、point 和平均 K 线数值仅是案例参数，不应推广到其他时期或品种。
- PDF p1553 的 10 pip 目标、10 pip 分批和 30–50 pip 止损是特定 EURUSD 案例，不构成通用参数；“此处没有加仓机会”说明图中价格并未实际到达全部计划层级。
- PDF p1555 中文加入“双顶、更高高点主要趋势反转”等图形标签且方向疑似混乱；p1556 英文明确为 `DT LH MTR`，即双顶、更低高点主要趋势反转。
- PDF p1536–1537、p1547、p1557–1558 为封面、目录、分节或结束页，不提供独立论据。
