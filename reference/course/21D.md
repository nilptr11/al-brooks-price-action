# 21D：反转期望值、TBTL 与双顶双底归类

来源：`SRC-COURSE-01-36/21D.txt`，对应 PDF 物理页 1843–1867。

## 内容结构

1. 用 40% 胜率、2:1 回报风险比演示主要趋势反转的期望值（PDF p1845–1849）。
2. 定义十根 K 线、两段走势（TBTL）及其正确时间周期（PDF p1851–1854）。
3. 重申趋势反转始于通道突破，并随后建立相反方向通道（PDF p1855–1856）。
4. 将大多数反转归为广义双顶/双底测试，并展开 HL、LL、HH、LH 反转的两段目标（PDF p1858–1864）。
5. 汇总通常涉及交易区间的反转形态家族（PDF p1865）。

## 核心思想

- 课程用保守模型解释低胜率反转仍可盈利：十笔类似交易中四笔各赚两倍风险、六笔各亏一倍风险，毛期望仍为正。Emini 示例是风险 2 点争取 4 点，外汇示例是风险 20 pip 争取 40 pip（PDF p1845–1847）。
- 其余约 60% 并不必然都是完整止损，课程称许多只是小亏或小赚、彼此抵消；少数赢家也可能远超初始两倍目标，因此实际结果可能好于简化模型（PDF p1845、p1848–1849）。
- 突破、高潮或反转后的最低目标常用 TBTL 表达：相反方向至少约十根 K 线、两段走势。它描述持续时间和结构，而不是固定价格距离（PDF p1851）。
- TBTL 必须以能够完整显示原形态的最高相关时间周期判断。如果十五分钟楔形同时在六十分钟图上成立，只完成十五分钟的十根两段仍可能太小，应检查六十分钟上的十根两段（PDF p1852–1854）。
- 所有趋势反转都先突破原通道/趋势线，但第一次通道突破可以只形成回调，原趋势仍恢复。真正反转还需要后来再次失败并形成相反方向通道（PDF p1855–1856）。
- 多头转空头的确认结构是空头突破后形成更低高点和更低低点；空头转多头则是多头突破后形成更高低点和更高高点（PDF p1856）。
- 课程把所有顶部广义视作双顶变体、所有底部视作双底变体，因为反转通常包含对旧极值、突破点或结构水平的再次测试。形态严重变形时，用主要趋势反转、失败突破、最终旗形或楔形描述更清楚（PDF p1858–1859）。
- 更高低点主要反转可视作双底：第一次上涨后回调测试先前空头低点，若强势向上反转，第二段上涨更可能出现，并可能成为一系列更高低点的起点（PDF p1860–1861）。
- 更低低点主要反转或微型双底同样以两段上涨为最低波段预期。课程建议第一目标至少减仓一半，完成 TBTL 和约两倍风险后，在多个阻力汇合处处理剩余利润（PDF p1862–1863）。
- 顶部完全镜像：更高高点反转后常再形成更低高点/双顶空头旗形，空头预期两段下跌；若首次反转本身已是更低高点，仍预期至少再一段下跌，有时发展成空头趋势（PDF p1864）。
- 主要趋势反转、最终旗形和其他失败突破、楔形、双顶双底、三角形、头肩、圆顶圆底及高潮大多包含一段交易区间。名称不同，核心都是测试、双向交易和随后突破（PDF p1865）。

## 重复与递进

- p1845–1847 用同一 40%/2:1 算术分别展示 Emini 与外汇，原则完全相同；p1848–1849 再说明实际亏损可能较小、赢家可能更大，是对保守模型的修正。
- p1851–1854 从 TBTL 定义递进到跨周期口径，并用十五分钟/六十分钟同一楔形重复说明最高相关周期优先。
- p1855–1856 重复 21A–21B 的“先突破旧通道，再形成新高低点序列”，本讲将其与 TBTL 和相反通道连接。
- p1858–1859 先给“所有反转都是双顶/双底”的总括，再承认外观常扭曲、其他术语更适合，后者是必要限制。
- p1860–1861 是同一 HL MTR 双底案例的拆分页；p1862–1864 扩展到 LL、HH、LH 及两段目标，形成结构映射。
- p1865 的形态清单再次说明多个名称共享交易区间和测试机制，是第 21 讲的汇总。

## 可能矛盾或边界

- 40% 胜率、2:1 回报风险比只是示范期望值。它假设每笔赢家、输家和执行成本符合模型；手续费、滑点、提前退出和尾部亏损都会改变结果。
- “60% 小盈小亏相互抵消”“胜率常高于 40%”“赢家常超过目标”是乐观修正，没有样本证据，不能用于保证实际结果优于模型。
- TBTL 是最低结构预期，不保证每次反转都完成。若反向压力不足，它可能只是很小回调并恢复原趋势。
- “最高时间周期”应是仍清楚包含同一形态的最高相关周期，不是无限向上寻找。周期越高，止损、目标和持有期也必须同步扩大。
- 第一次通道突破不等于趋势反转；p1855 的案例明确先恢复原趋势，后来另一回调才发展成反转。
- “所有顶部/底部都是双顶/双底”是极宽的分类约定。若把任何测试都算第二顶底，这一说法预测性很弱；实务仍需明确测试对象、信号和突破。
- TBTL 与 2 倍风险是两个不同目标维度，前者是时间/段数，后者是价格盈亏。二者未必同时到达，不应强行要求重合。
- 达到第一目标减半是管理示例，不是唯一最佳比例；剩余仓位要结合阻力、反向压力和账户计划。
- “实际风险更小”需要事后价格没有靠近止损，不能在入场时用事后实际风险替代初始风险计算。

## 翻译或抽取问题

- PDF p1845 的 60% 指主要趋势反转交易中未形成大赢家的部分，不是“60% 的强反转形态”有统一结果；标题中文容易混淆样本。
- PDF p1848–1849 的 `OT LH MTR` 应为 DT LH MTR 的 OCR 错误。
- PDF p1852–1854 的标题存在直接冲突：p1853 写“如果 K 线不足，查看更小周期”，但正文始终要求查看包含该楔形的更高 60 分钟周期。结合前后页，标题应视为错误或抽取异常，规则仍是最高相关周期。
- PDF p1853 中文开头把“60 分钟至少十根两段已满足”与标题混排，容易误认为要转向更小周期。
- PDF p1855 的第一次回调突破趋势线后多头趋势恢复，说明该次只是通道突破而非最终反转；图中真正 LH MTR 的位置需视觉信息。
- PDF p1859 的“计算机能识别变形双顶双底”是课程推断，没有算法证据，不应作为形态客观存在的证明。
- PDF p1860–1861 的“其他电脑把这里看作反转”无法由纯文本验证，只能保留多个可能起点的概念。
- PDF p1862 的 initial risk/actual risk 依赖课程另一个风险视频，仅凭本页无法展开或作外部核验。
- PDF p1864 图中 `OT` 是 DT 的 OCR 错误；HH、LH 两个卖点的先后位置依赖图示。
- PDF p1843–1844、p1850、p1857、p1866–1867 为封面、目录、分节或结束页，不提供独立论据。
