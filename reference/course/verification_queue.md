# 后续来源核对队列

本文件只定义以后如何核对，不在当前“提取和分析材料”阶段访问外部来源或裁决课程正确性。

## 一、来源层级

1. **同版英文原始课程**：对应视频、原始英文课件或官方字幕，用于确认原句、图表标签、页序和术语。
2. **Al Brooks 官方书籍/网站说明**：用于确认概念在其他官方材料中的稳定定义，但不能自动替代当前课件的具体表述。
3. **专业一手资料**：交易所、监管机构、经纪商合约说明、统计教材和期权风险资料，用于核对产品规格、执行机制和数学。
4. **实证验证**：对 80%、70%、40–60 等经验比例，需要预先定义市场、周期、样本、事件和目标；官方重复同一数字只能确认“作者这样主张”，不能替代统计验证。

每项核对应记录：课程位置、中文提取、英文原句、来源链接/版本、适用对象、是否翻译错误、是否只是教学近似、对仓库表述的影响。

## 六、执行顺序建议

1. 先核 P0 原文与图表标签，形成可靠英文—中文术语对照。
2. 再完成 P1 数学重算；数学错误不等待经验统计即可标明。
3. 对 P2 只先确认作者原话和限定词，再决定哪些值得设计实证。
4. P3 若缺少可观察证据，保留为解释模型，不需要强行证明才能理解课程主体。
5. 完成以上核对后，才把课程材料与仓库现有正文逐条比对并决定合并、精简或改写。
