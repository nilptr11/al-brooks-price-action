# 01–52 跨讲重复矩阵

## 判定标准

- **必要定义复现**：核心概念在新专题中被再次调用，保留引用，但无需重新完整解释。
- **牛熊/品种/周期镜像**：用于证明规则对称或展示执行差异，材料应保留，知识节点可共用。
- **规则—案例递进**：后文给出新条件、算例、失败分支或管理方法，不能因主题相同而删除。
- **低增量重复**：相邻页面或跨讲只换措辞，没有增加条件、证据或反例，成稿时应合并。
- **张力性重复**：重复过程中数值、对象或条件发生变化，必须并列并标注边界，不能直接合并为单一规则。

## 明确可精简的低增量位置

1. **03E p264–265**：损益换算完整重复页，无新增原则。
2. **17A p1400 与 p1403**：仅更换市场例子，核心“70% 胜率也可因差 RR 亏损”相同。
3. **31C p2463–2466**：连续四页重复“只取最小目标长期很难、需要极高胜率”，仅依次增加成本、例外和宽止损背景，可合为一段。
4. **43A–D 与 44A–D**：多空镜像程度极高；完整材料保留，结构化知识只需无方向母版。
5. **45A–E 与 46A–E**：宽幅多空通道镜像；定义和管理原则不应重复两遍。
6. **51B p1300–1302**：同一“良好管理可避免亏损”图例和句子连续重复，p1302 仅出现 OCR/术语差异。
7. **52A p1356–1357**：正文几乎相同，中文小标题却分别标“源于错误/源于坏运气”；不能据标题拆成两套思想。
8. **51C–52B**：实际止损、快速认错、退出后再入、止损远则减仓在四讲中高频复述；课程总结合理，仓库成稿应由一个风险管理节点承载。

## 不应因“重复”而删除的材料

- 牛熊镜像中的执行差异、翻译差异和止损方向。
- 同一规则在趋势、宽通道、普通区间和突破模式下的不同条件。
- 早入与确认后入的概率—风险—回报交换。
- 同一形态成功、失败、失败后再失败的路径更新。
- 日内案例中市场状态随时间改变的证据链。
- 课程数值发生变化或分母不清的重复；这些是需要核对的材料，而不是应被静默合并的冗余。
