# 22A：主要趋势反转的要求、压力与确认层级

来源：`SRC-COURSE-01-36/22A.txt`，对应 PDF 物理页 1868–1891。

## 内容结构

1. 给出顶部、底部主要趋势反转的两阶段要求（PDF p1871–1872）。
2. 对比买卖压力不足时的次要反转、旗形和交易区间（PDF p1873–1877）。
3. 说明 MTR 是低概率高回报的入场架构，而不是已经确认的反转（PDF p1879–1881）。
4. 展开强突破、跟进和新高低点如何把 MTR 从架构升级为新趋势（PDF p1882–1887）。
5. 比较 HH MTR 与 LH MTR 的入场选择，并解释更高低点处的多空共同买盘（PDF p1888–1889）。

## 核心思想

- 顶部 MTR 通常先有一次足够强的次要下跌，跌破多头趋势线；多头随后在旧高附近尝试恢复但失败，第二次向下反转若伴随强空头突破，才可能发展成空头趋势（PDF p1871）。
- 底部 MTR 完全镜像：第一次次要上涨突破空头趋势线，空头恢复在旧低附近失败，第二次向上反转伴随强多头突破，才可能建立多头趋势（PDF p1872）。
- MTR 的目标不是一小段逆势短线，而是至少两段和波段利润，成功后常先形成大型交易区间，只有一部分继续成为完整相反趋势（PDF p1871–1872）。
- 若最后一段逆势腿缺少足够买压或卖压，反转更可能只是次要反转：形成短线摆动、原趋势旗形或小交易区间，而不是主要反转（PDF p1873–1876）。
- 课程以约五至十根 K 线积累反向压力作为常见尺度。压力可由多根普通 K 线逐渐建立，也可由少数大实体、强收盘和跟进更快形成（PDF p1873、p1875，结合 21C 的量级说明）。
- 大多数 MTR 在交易区间中发展，需要多次尝试。课程在此明确给出：约 60% 的 MTR 只形成次要反转，约 40% 能形成波段交易（PDF p1877）。
- MTR 被称为最重要的反转架构，是因为它允许在新趋势完全确认前靠近极值入场，潜在回报相对止损较大；低约 40% 的成功率是获得这种回报结构的代价（PDF p1879–1881）。
- MTR 是 setup，不是已经发生的 reversal。交易者可在 HL/LH 架构处提前试仓，但只有后续突破、跟进和 Always In 转换才能确认反转（PDF p1881–1883）。
- 强突破本身仍需跟进。连续趋势 K 线、适当实体、收盘靠近极值和下一根跟进，才足以使多空双方接受新方向；确认后入场概率提高，但止损离当前价格更远（PDF p1882–1883、p1885）。
- 空头趋势线被强势上破后，多头关注更高低点并在向上反转处买入，希望它成为新多头通道一系列更高低点的第一个；多头趋势转空头则在跌破趋势线后等待更低高点卖出（PDF p1884、p1886）。
- 第一次突破虽好但跟进差时，市场仍可能只是交易区间。第二次强突破及跟进使旧趋势一方放弃，更可靠地启动相反方向（PDF p1885、p1887）。
- 顶部应在 HH MTR 直接卖出还是等 LH MTR，取决于此前空头腿强度。若第一次下跌弱，即使反弹也弱，最好等待更低高点；若已有强下跌和明显卖压，反弹到双顶/更高高点时也可尝试提前卖出（PDF p1888）。
- 底部强突破后的回调中，被困空头希望回调以较小亏损回补，多头希望更低价格买入，双方买盘共同形成更高低点并支持第二段上涨（PDF p1889）。

## 重复与递进

- p1871–1872 是顶部和底部 MTR 的完整镜像，适合合并成“趋势线突破—旧趋势恢复失败—第二次反转强突破”的三阶段模型。
- p1873–1876 以买压不足、卖压不足镜像重复次要反转，p1874、p1876 是前页结论的独立文本重复，可合并。
- p1877 把此前 40%/60% 口径明确映射到“波段/次要反转”，是对第 21 讲概率冲突的重要澄清。
- p1879–1881 从 MTR 的期望值、形态变体到“架构而非反转”逐步说明为何可提前入场。
- p1882–1887 多次对比弱突破与强突破跟进；新增层级是 Always In 转换、HL/LH 建立及旧趋势一方最终放弃。
- p1888–1889 分别讨论提前 HH/LH 选择和强突破后 HL 的订单机制，是执行层补充。
- 本讲系统重述 21A–21D 的主要反转思想，新增最清楚的 MTR 要求、40% 波段/60% 次要口径，以及“setup 不等于 reversal”的术语边界。

## 可能矛盾或边界

- 趋势线突破只是第一阶段，旧趋势恢复失败和第二次强反转同样必要。把第一次逆势腿直接叫主要反转会忽略 60% 次要分支。
- “第二次反转”不是机械计数任意两个信号，而是第一次趋势线突破后，旧趋势在极值附近恢复失败，再出现新的反向尝试。
- 约 40% 成功在本讲明确指形成波段交易，不等于 40% 都成为完整相反趋势；这可与 21C 的约 20% 大反向趋势并存。
- 21A 所称“40% 成为 major”仍有术语张力，可能把足够波段的 MTR 交易称为 major，而非要求最终形成大相反趋势。
- 五至十根 K 线只是压力积累的常见时间，一两根异常强突破也可达到类似效果；反之，十根重叠小 K 线未必有足够压力。
- MTR 良好风险回报依赖接近结构极值入场和止损有效，不能仅凭形态名称假设回报一定超过风险。
- 强突破确认后“风险更大”指止损距离更远；通过降低仓位，账户风险仍可固定。
- HH MTR 可以提前卖出的条件是已有强空头腿。若卖压不足，等 LH 能避免在原多头趋势恢复时过早做空。
- 强突破后形成更高低点/更低高点是新通道尝试，不保证系列一定延续；坏跟进仍可回到交易区间。

## 翻译或抽取问题

- PDF p1871 的“wedge bull channel too tight for HH MTR”表示紧密多头通道中提前 HH 反转概率低，图中具体楔形层级无法纯文本恢复。
- PDF p1872 的 `Test should be at least 33% PB toward low` 语义不够清楚。可能指旧低测试需从第一次上涨高点回撤至少约三分之一，当前不能稳定改写为通用 MTR 定义，应保留待核。
- PDF p1873–1876 的“五至十根”没有强度、连续性和时间窗口定义，只是课程经验尺度。
- PDF p1877 的 60% 与 40% 英文对应清楚：60% 次要反转，40% 波段；中文排版把两者位置打散。
- PDF p1880 只是形态标签图，标题“多数反转是 MTR 变体”来自中文，英文可提取文本没有完整解释句。
- PDF p1881 中文“主要趋势反转最小化风险并最大化利润”应理解为潜在结构优势，不是结果保证。
- PDF p1882 图中 HLMTR、LH MTR 与 Always In 标签位置依赖图形，不能仅凭抽取文本重建先后。
- PDF p1885 中文“首次突破后弱势跟进显示正在做空”词序不清；英文是首次空头突破跟进差，交易者尚不相信新空头趋势。
- PDF p1888 中文一处把 LH MTR 写成“更低低点主要趋势反转”，应为更低高点。
- PDF p1889 的“双方都买入”分别是空头回补与多头新买，经济动机不同但订单方向相同。
- PDF p1868–1870、p1878、p1890–1891 为封面、目录、形态清单或结束页，不提供新增独立论据。
