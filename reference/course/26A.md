# 26A：三角形的五腿定义、成熟度与旗形/反转职责

来源：`SRC-COURSE-01-36/26A.txt`，对应 PDF 物理页 2118–2140。

## 内容结构

1. 定义三角形为至少一个方向有三推、总体至少五腿的紧密交易区间（PDF p2119–2122）。
2. 说明收缩、上升、下降三角形的突破模式、磁吸和近似 50/50 方向（PDF p2123–2127）。
3. 将楔形放在“倾斜三角形”的连续谱上，比较 50% 与 75% 方向先验（PDF p2128–2130）。
4. 区分三角形作为趋势顶部/底部反转或原趋势旗形，并展开入场确认层级（PDF p2131–2134）。
5. 说明腿可有多种计数方式，形态大小和形成时间会决定是否已中和原趋势优势（PDF p2135–2138）。

## 核心思想

- 三角形是横向的紧密交易区间，至少一个方向出现三次推动，因此完整交替结构最低通常有五腿。它可以收缩、上升、下降或扩张；若整体明显倾斜，课程更倾向称为楔形（PDF p2121–2122）。
- 三角形与 TTR 的交易方法相同。三推很清楚时可叫三角形，不清楚时直接叫紧密交易区间；名称不改变其双向突破和失败突破逻辑（PDF p2122）。
- 交易区间具有磁吸，突破容易被拉回。课程把成熟三角形视为突破模式，向上、向下大致各 50%，并把突破交易结果近似描述为一半获利、一半反转（PDF p2123–2126）。
- 原趋势可能保留轻微优势，尤其三角形形成不久时。空头趋势中的三角形可略偏向顺势向下，但随着区间持续约二十根以上，课程认为原趋势优势基本消失，方向接近 50/50（PDF p2123–2125、p2137）。
- 上升三角形下边界上升、上边界近似水平；下降三角形上边界下降、下边界近似水平。课程不同于常见名称暗示，仍将成熟两者都视为上下突破近似等概率（PDF p2124–2125）。
- 有效向上突破后，多头可在突破回调买入，被困空头也会回补；向下突破完全镜像。执行重点是突破质量与回调，而不是预测三角形名称指向的方向（PDF p2126）。
- 三角形尺度可以从 ii/iii 连续内包 K 线到大型多腿区间。ii/iii 在低周期展开为三角形，所以当前周期看不到五腿也可按低周期结构理解（PDF p2127）。
- 楔形是向上或向下倾斜的三角形。越接近横向三角形，突破方向越接近 50/50；越像清楚的楔形通道，课程越倾向使用约 75% 向通道反方向突破的先验（PDF p2129–2130）。
- 三角形既可能是趋势末端反转，也可能是趋势延续旗形。外部趋势、区间持续时间、反向压力和最终突破决定职责，不能仅凭五腿结构预先定性（PDF p2132–2134、p2137–2138）。
- 顶部案例展示确认交换：区间内提前卖出概率低但止损近；弱空头突破后连续约八根没有阳线实体，市场转为 Always In Short，卖出概率提高但止损更远；最强确认点风险距离也最大（PDF p2132）。
- 三角形可能有七腿、九腿，而非严格五腿；每个大腿又常可细分为两个更小腿。不同计数者可采用合理的不同分段，只要最终对市场状态和执行判断一致（PDF p2133、p2135–2136）。
- 空头趋势中的多次向下尝试若在第 3、5、7、9 腿持续失败且逐次变弱，三角形可从空头旗形转为底部反转；向上反转和突破回调提供买入确认（PDF p2134）。
- 非常紧、形成时间短的多头三角形还不足以把多头优势降到 50%，应先视为多头旗形；但若位于趋势晚期，也可能成为最终多头旗形，突破延续后再形成顶部（PDF p2137）。
- 中等大小三角形若仍在空头趋势、阳线数量和买压不足，更可能是空头旗形而非 MTR；HL 只产生次要反转和横盘，空头突破回调后原趋势恢复（PDF p2138）。

## 重复与递进

- p2120 再次重复反转形态家族清单，只把焦点切到三角形，没有新增机制。
- p2121–2122 从几何/腿数定义递进到 TTR 功能定义，说明三角形只是更清楚的紧密区间。
- p2123–2126 多次重复交易区间磁吸、突破模式和近似 50/50；p2124–2125 的上升/下降形态用来否定名称本身的方向优势。
- p2127 把 23B 的 ii/iii 最终旗形重新归类为微型三角形，新增大小尺度而非新交易规则。
- p2129–2130 重复第 24 讲的“楔形是倾斜三角形”，但首次较清楚地把横向 50% 到倾斜 75% 表述为连续谱。
- p2132–2134 从顶部案例、一般 50/50 到多次失败下破的底部反转，展示三角形如何在旗形和反转之间转换。
- p2135–2136 只有计数方法差异，核心重复“不同标签/腿数仍可采用同一交易判断”。
- p2137–2138 是多头旗形与空头旗形的环境例子，新增关键变量是形成时间是否足以使原趋势概率回到中性。

## 可能矛盾或边界

- p2126 称三角形“总是突破模式”，p2137 又说非常紧、K 线不足的三角形“还不是交易区间和突破模式”。这是直接表述张力。较合理的边界是：成熟三角形是中性突破模式；刚形成的小旗形仍保留明显顺势优势，尚未达到 50/50。
- p2123 一面说顺原趋势突破概率略高，一面写突破获利/反转各 50%；p2124 又以约二十根作为回到 50/50 的条件。文档应按区间成熟度区分，不能把三句当成同一时点的固定概率。
- “二十根以后任何区间上下各 50%”是经验尺度，不是第 20 根发生离散概率跳变。突破压力、位置和高周期环境仍会造成偏差。
- p2123 说突破有 50% 获利、50% 反转，又说因磁吸“通常在五至十根内失败”。若“通常”指所有突破则与 50/50 有词义张力；更合理地理解为失败突破通常会在五至十根内暴露，而非超过一半的所有突破必然失败。
- 向上/下降三角形仍给 50/50，与传统形态名称常暗示方向不同。这里应忠实记录课程观点，但不能在没有课程样本的情况下写成经过统计验证的事实。
- 当前周期 ii 只有两根内包 K 线，却被称为最低五腿三角形，是跨周期定义：五腿存在于更小周期，而非当前周期可见。若文档不写周期层级会显得自相矛盾。
- p2129–2130 说分不清楔形或三角形“不重要”，但又分别赋予约 75/25 与 50/50。名称本身可不重要，倾斜程度和行为先验却仍重要；两者不应混为一谈。
- 从 50% 到 75% 的连续谱没有给出斜率、收敛度或推动强度的量化映射，只能作为定性调整，不能精确插值。
- 三角形可有五、七、九腿且每腿还能细分，会带来主观计数和多重尺度。计数应服务于“多次失败尝试”判断，不能事后任意重分以匹配形态。
- p2132 把弱空头突破后八根无阳线实体视为 Always In Short，说明持续时间可补偿单根突破弱；但“最高概率卖点”仍需图中位置，不能仅由文本泛化为任意八根。
- p2134 的“更低价格没有卖家、市场不得不上涨”是参与者/目的论叙事；可观察证据只是多次下破失败、每次卖压减弱及最终向上突破。
- 趋势晚期的小旗形既可顺势突破又可成为最终旗形。`final` 仍是事后身份，不能因位置晚就提前确定反转。

## 翻译或抽取问题

- PDF p2121 的“至少五腿”依赖交替推动计数；正文 p2122 又说至少一个方向三腿，二者应合并理解，不是两个独立定义。
- PDF p2123 中文“50% 的获利机会、50% 的反转机会”没有说明获利目标；不能等同于完整测量移动或趋势成功率。
- PDF p2123 的 `2–3`、五至十根失败和趋势晚期最终旗形标签依赖图形，纯文本无法复原具体突破。
- PDF p2127 的 `ii`、`iii` 分别指连续两根、三根内包 K 线，不是章节编号。
- PDF p2129 中文把 75% 句子拆成两处；英文含义是楔形空头通道约 75% 作为多头旗形向上突破，但常先有一次失败的空头突破。
- PDF p2132 的 `OT` 很可能是 `DT`；三个卖点的概率和止损比较依赖图中位置，文本只能恢复早/中/晚确认层级。
- PDF p2133 的 `Ot_` 是图形/OCR 残片，不影响五、七、九腿结论。
- PDF p2135 没有可用正文，只有标题；替代计数的内容全部来自 p2136。
- PDF p2136 的中文“具有双重底且更高低点的 MTR”对应图中 `Triangle or DB HL MTR`，不能确定三角形与 DB 是并列标签还是嵌套结构。
- PDF p2137 的三角形顶、最终旗形顶点测试和 Buy The Close 阶段依赖图形先后；纯文本可恢复总体序列，无法定位精确 K 线。
- PDF p2138 标题缺失，买压不足、次要反转和空头旗形内容来自图内标注。
- PDF p2118–2119、p2128、p2131、p2139–2140 为封面、目录、分节或结束页，不提供新增独立论据。
