## 怎样处理内容冲突

1. Brooks 概念或术语冲突，先回到正式来源和对应学习专题。
2. 策略条件冲突，以对应策略页为准，不从学习正文拼接临时计划。
3. 订单、成交、保护或结束流程冲突，以执行手册为准。
4. 策略地图发现缺口时，先判断该主题是否真的能够形成独立完整计划；不能独立说明入场、止损、目标和管理的内容，不强行建立策略。

来源支持价格行为关系，仓库负责把这些关系整理成可读、可执行的人工流程。策略分类和执行基线仍然是仓库整理，不应写成 Brooks 官方穷尽分类。
