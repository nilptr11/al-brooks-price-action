# 参考与来源

本目录保存术语快速查阅、官方来源登记和主题审计。它负责证据与可追溯性，不直接规定交易动作。

| 文件 | 职责 |
| --- | --- |
| `glossary.md` | Brooks 通用术语快速解释；非规范，不收录执行手册的版本化内部字段。 |
| `official_sources.md` | 官方来源 ID、链接、访问范围和核验日期。 |
| `audit_matrix.md` | 主题、执行规则、学习文档、官方来源和审计状态的对应关系。 |
| `audits/` | 专题审计，例如 pattern atlas 的定义和几何验收。 |

权威关系：

- 官方资料是证据权威。
- 当前[执行手册](../README.md#权威层级)是仓库执行权威。
- 本目录发现冲突时先记录证据和状态，再修改执行手册或学习文档。
