# Context、位置和控制权

> **状态：Learning / Navigation**
>
> 本页只组织本章范围、前置条件和学习顺序，不定义交易动作。

## 章节目标

把 market cycle 放回具体位置、控制权、周期和时段中，判断一个 setup 是否处在有意义且目标空间足够的环境。

## 前置条件

先能区分 trend、trading range 及 breakout / channel 的基础层级。

## 完成标准

能够从累积 buying / selling pressure 说明控制方向、关键位置和路径障碍，并把常见价格行为倾向与需具体市场验证的时段判断分开。
