# Setups

> **状态：Learning / Navigation**
>
> 本页只组织本章范围、前置条件和学习顺序，不定义交易动作。

## 章节目标

先理解 setup 如何把 context、pattern 和入场依据组合起来，再按需比较趋势 flag、突破、区间交易、反转和二次入场。

## 前置条件

先完成 market cycle 与核心 context 阅读；不要求预先背完整形态库。

## 完成标准

能够分清 pattern、setup 与完整交易，并为具体 setup 说明结构、entry、合理 price-action stop、profit target 和管理方式，而不是只报形态名称。

每一类 setup 都按同一顺序阅读：

```text
context -> setup 结构 -> entry -> reasonable stop -> setup target -> management
```

目标不是按形态名称套固定公式，而是结合 setup 的价格行为预期、market cycle、support/resistance magnets 与 scalp/swing 选择确定，具体顺序见[什么是 Setup](00_what_is_a_setup.md#目标如何确定)。

## 快速比较

| Setup | 核心结构 | 常见价格行为预期 | Stop 依据 | 常见 target 依据 |
| --- | --- | --- | --- | --- |
| 趋势延续 | 既有趋势 → 回调未破坏主要 swing → 原方向恢复 | H2/L2 通常较快恢复；常被 scalp，趋势仍强时可 swing。Moving-average gap bar 是常见的旧极值测试 setup | Swing 使用正确 price-action stop；强趋势常参考最近 major HL/LH，具体位置随结构变化 | 合格 scalp、旧极值、前高低、趋势/通道线；结构清楚时才加入 measured move |
| 突破延续 | 清楚边界 → 强突破 → 跟进或回踩守住 → 新区域被接受 | Breakout 通常至少继续一点；强 breakout 常支持 measured move，回调后至少第二腿是一般预期 | 强 breakout 的初始 stop 可在完整 breakout leg 外，之后可随新 major HL/LH 推进 | 区间/形态高度、breakout 大小、Leg 1 = Leg 2 与前方 magnets |
| 区间 fade | 成熟区间边缘 → 突破失败 → 回到旧区域 → 区间内轮动 | Buy low, sell high, scalp；下半区多头通常在上半区兑现，空头镜像 | Limit/scale-in 使用 reasonable stop，且有时必须很远；确认型 stop 另按其价格结构评估 | 合格 minimum scalp、实际风险、区间内部支撑阻力与上/下半区 |
| MTR | 成熟趋势 → 通道突破 → 旧极值测试失败 → 反向 swing | 成功结果是两腿和 swing profit，常先形成较大交易区间；TBTL 是一般路径预期 | 早期 stop 参考旧极值测试/完整反转；确认后概率较高但 stop 通常更远 | 早期低概率语境常以约 2R 作 swing 对照；目标仍来自反向路径 magnets，结构清楚时才加入 measured move |
| Second entry / trap | 第一次 entry → 回撤或失败 → 同逻辑第二次触发，或反方失败后另建 setup | Second entry / trap 本身没有独立目标；价格预期由所在趋势、突破、区间或 MTR 决定 | 使用上层 setup 的 price-action stop | 继承上层 setup；计数本身不生成目标 |
