# 学习路径

> **状态：Navigation / Non-normative**
>
> 本页只安排阅读顺序和专题入口，不定义具体入场、止损或执行动作。

## 核心路径

按顺序完成七份材料：

1. [Al Brooks 方法主线](00_method/00_al_brooks_thesis.md)：market cycle、context、setup 与 Trader's Equation。
2. [市场周期](01_market_cycle/00_market_cycle.md)：trend、trading range、breakout 与 channel。
3. [背景、位置与控制](02_context/00_context_location_control.md)：判断双方力量和目标空间。
4. [什么是 Setup](05_setups/00_what_is_a_setup.md)：区分 pattern、setup 和完整交易计划。
5. [接受与失败](03_order_flow/00_acceptance_and_failure.md)：理解跟进、失望、交易前提和失败。
6. [概率、风险与回报](00_method/01_probability_risk_reward.md)：使用 Trader's Equation 比较机会。
7. [交易计划](06_trade_management/00_trade_plan.md)：把入场、止损、目标、仓位和管理连成一体。

完成后，应当能够用自然语言说明：

- 当前市场处于什么周期；
- 价格位于什么重要位置；
- 哪一方更可能控制，反方理由是什么；
- 当前 setup 交易的市场行为是什么；
- 合理的保护止损和现实目标在哪里；
- 什么新信息会削弱或否定原判断。
