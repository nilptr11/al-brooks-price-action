# 交易管理

> **状态：Learning / Navigation**
>
> 本页只组织本章范围、前置条件和学习顺序，不定义交易动作。

## 章节目标

把 setup、protective stop、profit target、仓位和退出放进同一笔交易。

## 前置条件

先理解 setup、接受/失败证据和 Trader's Equation；本章在这些基础上补全仓位、目标、退出和成交后管理。

## 完成标准

能够为具体 setup 说明 entry、reasonable stop、profit target、仓位和管理，并用 Trader's Equation 检查它们是否匹配。

## 本章文件

1. **核心** [`00_trade_plan.md`](00_trade_plan.md)：把 entry、失效、stop、目标、仓位和管理连成同一方案。
2. **按需** [`01_scalp_vs_swing.md`](01_scalp_vs_swing.md)：分清 scalp 与 swing，并理解 TBTL 的时间和腿数预期。
3. **按需** [`02_scaling_in_out.md`](02_scaling_in_out.md)：学习加减仓及其总风险；掌握基础 stop-entry swing 后再练习。
4. **按需** [`03_risk_psychology.md`](03_risk_psychology.md)：用仓位、连续亏损和行为纪律检查执行稳定性。
