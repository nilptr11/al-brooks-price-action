# 接受、失败与订单方式

> **状态：Learning / Navigation**
>
> 本页只组织本章范围、前置条件和学习顺序，不定义交易动作。

## 章节目标

用价格行为判断市场是否接受一次尝试，分清 entry disappointment、premise 变化、failure，以及入场单与保护性止损。

## 前置条件

先理解 market cycle、context、setup、signal bar 与 entry bar；订单术语不清时先查[核心术语表](../../reference/glossary.md)。

## 完成标准

能够区分弱跟进、premise 变化和 failure，并能说明 stop entry 与 protective stop 的不同用途。
