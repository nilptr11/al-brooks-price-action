## 章节目标

先区分 trend 与 trading range，再理解 breakout、tight channel、broad channel、breakout mode 和 climactic behavior 如何描述市场状态及其转换。

## 前置条件

先读[方法与读图主线](../00_method/00_al_brooks_thesis.md)，知道形态标签不能替代 context 和交易数学。

## 完成标准

能够对一段隐藏未来的走势给出当前 market cycle，并指出哪条新证据会维持或改变判断，而不是只贴静态标签。
