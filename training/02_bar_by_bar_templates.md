# 逐根观察模板

> **状态：Training / Derived**
>
> 本模板只保存训练观察；交易计划、订单、成交和管理事件统一使用当前[执行手册](../EXECUTION_MANUAL.md)的[核心交易计划](../EXECUTION_MANUAL.md#4-一份核心交易计划)。

## 使用边界

- 每次只训练一个问题，并隐藏尚未发生的 K 线。
- 每揭示一根 K 线就追加一行；过去的行保持只读，不用后续信息改写。
- 可观察事实与判断分开记录。没有实质新增信息时可以写“无变化”。
- 本轮问题包含计划设计时，出现可设计的交易想法后另行填写执行记录；纯观察轮可以停在当时判断，不把候选强制升级为计划。本模板只引用已建立记录及其时间，不复制任何计划字段。
- 本轮问题包含订单或持仓管理时，相关状态只更新在同一执行记录。观察表可以写“见执行记录及时间”，不重写状态、数量、价格或动作。
- 只有后续结果才能确认的名称，不得写回先前观察。

## 训练信息

```text
训练日期：
品种 / 周期 / 时段：
回放或复盘区间：
本次只训练的问题：
对应执行记录（如已建立）：
```

## 逐根观察时间线

| 新增事实（含 K 线或时间） | 判断变化 | 待验证信息 |
| --- | --- | --- |
|  |  |  |

- **新增事实**：只写这一根新出现且当时可见的价格行为。
- **判断变化**：写“保持、增强、削弱、失效或无变化”，并说明影响的是市场结构、控制权、位置或当前交易想法；如已影响规范记录，只附记录时间。
- **待验证信息**：写下一根或下一段需要出现什么，才能区分当前仍未决的路径。

## 完成标准

训练结束时，时间线应能仅凭当时信息重建判断如何变化，同时不会形成第二份交易计划或订单日志。纯观察轮不因没有计划或订单而不合格；随后使用[复盘检查清单](03_review_checklist.md)产出流程结果、市场结果、主要问题和下一步学习。
