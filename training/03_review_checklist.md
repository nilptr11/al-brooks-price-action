# 交易复盘检查清单

> **状态：Training / Derived**
>
> 本清单用于训练和复盘，不增加价格行为定义。Market reading、setup、stop、target 和 Trader's Equation 回到相应学习内容与来源；计划、订单、成交和管理事实来自当前[执行记录模板](../EXECUTION_RECORD_TEMPLATE.md#单笔交易记录)，仓库订单与记录操作见[执行手册](../EXECUTION_MANUAL.md#7-仓库订单与记录操作)。

## 使用方式

每次先复盘一段连续走势，再逐笔核对本段建立的交易记录。先锁定当时可见的信息，再查看后续结果；若使用逐根回放，先完成[逐根观察模板](02_bar_by_bar_templates.md)。本清单只引用原始观察、批次执行配置和交易记录，不重抄计划。

## 九个核心问题

1. 当时记录是否完整保留，且没有用后续走势覆盖原判断或原计划？
2. 当时在决策周期上对 market cycle、控制方向和主要位置的判断是什么；多空双方分别有什么理由？高周期若被使用，是否写清它具体改变了位置、障碍、目标或不确定性？
3. 当时记录的支撑阻力、gap、EMA、breakout point 或 measured move 中，哪些实际影响了入场、正常测试、失效、路径障碍或目标；它们是否在结果出现前已有明确价位、来源周期和作用？
4. 若准备交易，setup、entry logic、signal bar 或结构依据、偏移 tick、触发价、平台订单类型、允许成交范围和撤单条件是否相互一致？若没有交易，当时记录的理由是什么？
5. Reasonable protective stop、profit target、scalp/swing 管理、仓位和 Trader's Equation 是否描述同一笔交易？费用和滑点如何影响实际结果？
6. 计划完成、活动订单、模拟或真实成交是否被正确区分；条件改变时是否撤销受影响的旧订单、保留旧记录，并按新信息建立新计划？
7. 每次成交后，实际或模拟持仓是否立即获得完整保护；部分成交时，持仓与未成交余单是否分别处理？
8. 持仓后是否根据新的 price action 区分正常波动、target 测试和 premise 变化；stop、target、scale-in/out 或退出的改变是否有当时价格证据和相应风险计算？
9. 本轮流程复盘重点在连续观察和自然出现的候选中执行得怎样；流程最早在哪一步结束或偏离，哪一条当时记录能直接证明？

## 最终复盘输出

### 1. 流程结果

只选择一个最符合当时过程的结果，不按最终盈亏归类：

- **观察完成**：本段连续走势记录完整，没有准备交易 setup，或预先明确为纯观察。
- **计划后未提交**：分析了一个 setup，但根据当时 context、stop、target、Trader's Equation 或个人能力决定不提交订单。
- **订单未成交**：已经提交模拟或真实订单，最终没有成交。
- **执行完成**：计划、订单、成交与管理事件都有可复核记录；该分类不评价最终盈亏或 setup 是否“应该”成功。
- **记录不完整**：准备承担风险或已经产生订单/成交，但缺少还原当时交易所需的信息。
- **执行偏差**：原计划可以复核，但订单、仓位、保护、目标或管理没有按其执行。

```text
流程结果：
支持该归类的当时记录：
流程结束或首次偏离的位置：
```

### 2. 市场结果

市场结果只记录后来发生了什么，用于核对假设和实际成本；它不能反向修改流程结果、决策快照或价格地图。

纯观察轮没有执行记录时，订单、成交和盈亏字段写“不适用”，只记录后来市场如何发展。

```text
执行来源 / 是否成交 / 最终成交数量：
触发后获得跟进、失败、回到公平区域，还是形成持续反向运动：
原计划中的目标、失效、跟踪止损或时间条件后来如何发展：
退出原因和成交结果（如有）：
毛结果 / 从配置或成交记录得到的成本 / 净结果（如有）：
```

零成交或未提交订单的样本也可以记录后续市场路径，但“后来走对了”不能改写当时记录。

### 3. 主要问题

一次只选择一个最值得修正的问题。问题必须引用当时记录并落到具体步骤，不使用“感觉不好”“不够耐心”等无法复核的总结。

```text
主要问题（若无流程问题，写最需要继续验证的不确定点）：
所在步骤：观察记录 / market cycle 与 context / setup 与计划 / Trader's Equation / 订单与成交 / 持仓管理
当时证据：
原要求：
实际偏差：
```

### 4. 下一步学习

把主要问题转换成下一轮的一个流程复盘重点，再回到[训练闭环](README.md#一次完整训练)。不要求下一段市场出现对应场景；没有出现时继续观察，不主动搜索。

```text
要复习的文件或概念：
下一轮流程复盘重点：
从哪个历史位置继续：
用什么当时记录证明已有改善：
```

## 被动样本索引

复盘结束后可以从当时的决策快照和价格地图生成简短索引，例如决策周期、market cycle、执行尺度、setup，以及实际影响计划的 gap、EMA 或 measured move。标签只用于以后筛选自然积累的样本；若复盘发现当时分类缺乏证据，记录“当时分类证据不足”，不能用结果重写原标签。已经看过结果后再次回放的样本属于纠错练习，不计作未知未来的盲测样本。
