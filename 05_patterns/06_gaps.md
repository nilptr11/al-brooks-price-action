# Gaps

## Gap Up / Gap Down

Gap up 表示市场高于前一区域开盘，gap down 则相反。缺口说明市场在开盘前或两段价格之间已经重新定价。

缺口后有两种基本可能：

- 市场接受新价格，继续朝缺口方向运动。
- 市场拒绝新价格，尝试回补缺口。

判断关键不是缺口本身，而是缺口后的跟进。

## Body Gap 和 Micro Gap

Body gap 指相邻 K 线实体之间留下空隙。Micro gap 指很小但可见的不重叠或跳跃。它们常说明短期订单压力强。

在强趋势中，连续 body gap 或 micro gap 可以显示趋势仍有惯性。在交易区间边缘，如果缺乏跟进，也可能变成诱多或诱空。

## Breakout / Measuring / Exhaustion Gap

Breakout gap 表示市场跳过旧边界并进入新区域。如果有跟进，缺口可能成为支撑阻力。

Measuring gap 可作为目标测算线索，但不是保证。

Exhaustion gap 发生在趋势后期。它看起来很强，但后续缺乏跟进，可能表示最后一批追随者入场。

## Failed Gap

Failed gap 是市场开在新价格区域后无法继续，并回到原区域。追随缺口方向的交易者可能被困。
