# 趋势延续策略

本目录只收录原趋势仍然控制市场时的顺势计划。先确认趋势，再选择回调的类型；不能因为看见 H1、H2、楔形或均线附近的 K 线，就反过来证明趋势存在。

**来源层**：强趋势第一次回调、H1/H2、wedge flag、moving-average gap bar 和旧极值测试等市场行为来自 Brooks；各页的精确 K 线外观、根数窗口、两根订单寿命、止损锚点、单一旧极值目标与全量退出，是仓库执行基线，不是 Brooks 对这些形态的统一参数或唯一管理。

本页只在[全局选择状态机](../how_to_choose.md)路由到趋势家族后，负责本家族内部分类与重叠优先级；具体资格、触发、参数和特有失效以所选策略页为准。本页不改写时段覆盖、单候选、结束或重入规则。

## 先判断是否属于本目录

只有同时满足以下条件，才从本目录选择策略：

- 当前趋势判断已经绑定交易周期、可见观察窗口和外层背景，没有用小周期局部 trend 代替大周期状态；
- 当前方向已经通过连续推进、强收盘、浅回调或反方失败表现出控制力；
- 最近的回调尚未破坏决定趋势的主要摆动点；
- 价格不在成熟交易区间的中部；
- 从计划入场到策略目标仍有足够空间；
- 合理的价格行为止损没有超出事先确定的风险边界。

如果回调越来越深、重叠明显增加、双方都能持续获利，先把市场重新判断为宽通道、交易区间或转换阶段，不要强行套用趋势延续策略。

## 状态不清时怎样处理

逐层 fallback 只处理当前没有把握的层级，并且只是保守管理假设，不自动生成入场资格：

- 连 trend 与 trading range 都分不清时，先按 trading range 管理并继续观察；只有某张区间策略自己的成熟边界、位置和触发完整时，才可进入；
- 已确认是 trend，但分不清 breakout phase 还是 channel 时，先按 breakout phase 管理，避免过早逆势；
- 已确认是 channel，但分不清 tight 还是 broad 时，先按 tight channel 管理；
- 分不清 broad channel 还是 trading range 时，先按 trading range 管理。

强 breakout phase 的第一段可识别回调会结束当前这段 spike 并进入 channel，但不结束整个 trend。回调覆盖前一突破点、相邻腿开始重叠时，应记录 staircase；连续延伸缩小或局部区间反复迁移时，还要检查 shrinking stairs 或 trending trading range。它们可能仍有方向，却说明双边交易增加，不能把“仍在趋势”和“仍适合强趋势策略”当成同一结论。完整选择顺序以[如何选择并锁定一项策略](../how_to_choose.md)为准。

## 同一次回调只选择一套计划

同一段回调可能同时被描述成 H2、双底、楔形或均线测试，但只能选择一张策略页。在泛化页的入场信号完成以前，先检查带固定参照的特异页是否已经出现其区别性事件，例如均线反侧完整 K 线、连续二十根不触均线、再次远离确认或清楚三推；区别性事件先出现时，继续按特异页等待，不得选择泛化页来取得更远目标、更窄止损或更有利解释。

这里的“候选锁定”是指按照[如何选择策略](../how_to_choose.md)第一次只保留一张候选页，并把所选页面、待确认事实和放弃条件写入记录的时点；它不是订单成交时点，也不能在结果出现后倒推。

第二次均线缺口是明确边界：如果[第二次均线缺口回调](second_ma_gap_pullback.md)第四步“再次远离确认”在普通[第二次恢复信号](second_signal_continuation.md)完成以前已经成立，后续第二次信号只使用第二次均线缺口页。反之，普通二次恢复信号已经先完成并锁定计划，而当时尚无合格再次远离确认，才保留普通二次恢复信号策略；后来不得用结果补画均线缺口序列来改名。

其他特异事件页与泛化页重叠时使用同一先后原则：在共同入场信号完成前已经出现特异页的区别性事件，就使用特异页。若两张特异页同时合格，而当时信息仍不足以唯一说明主要交易前提，就继续等待或放弃，不挑选目标更远或止损更近的一页。计划写定后，不因后来出现更漂亮的名称而切换入场、缩小止损或延长目标。

如果原计划在成交前失效，先取消并确认订单已经结束；只有新的价格过程独立满足另一张策略页时，才重新建立计划。如果已经成交，先按原计划处理所选环境确认的仓位，不能把策略改名来逃避止损。

## 趋势家族特有约束

价格、净 R、等待失效、保护与固定管理的通用语义只由[策略共同规则](../common_rules.md)定义。下列内容只补充趋势家族的选择和目标边界：

- 做多与做空使用完全镜像的判断；
- 各页写定的旧趋势极值，是本目录把“测试旧高／旧低”固定成可执行价格的基线目标，不表示 Brooks 认为强趋势只能在旧极值退出，也不覆盖 swing runner 或按主要 higher low／lower high 移动止损的管理方式；若要使用另一套 swing 管理，必须在候选形成前选择独立计划，不能成交后延长本页目标；
- 各页的结构止损与固定目标构成机械订单基线，但标准计划仍继承[共同 premise-change 规则](../common_rules.md#等待失效与保护基线)：普通反向 K 线或小回测不自动退出；强反向突破与跟进、外部价格持续接受或清楚的 Always In 翻转否定原 premise 时，按主动全量退出流程处理。只测试 stop / target 的回放必须另建 `mechanical_only` 独立研究变体，不得并入标准结果；
- 目标已经被入场价越过，或途中存在更近且无法合理穿越的强障碍时，放弃；
- 看不清趋势、回调顺序或主要摆动点时，放弃；
- 订单、成交、所选环境确认的仓位和保护操作统一遵守[人工交易执行手册](../../execution/execution_manual.md)。
