# 趋势延续策略

本目录只收录原趋势仍然控制市场时的顺势计划。先确认趋势，再选择回调的类型；不能因为看见 H1、H2、楔形或均线附近的 K 线，就反过来证明趋势存在。

## 先判断是否属于本目录

只有同时满足以下条件，才从本目录选择策略：

- 当前方向已经通过连续推进、强收盘、浅回调或反方失败表现出控制力；
- 最近的回调尚未破坏决定趋势的主要摆动点；
- 价格不在成熟交易区间的中部；
- 从计划入场到策略目标仍有足够空间；
- 合理的价格行为止损没有超出事先确定的风险边界。

如果回调越来越深、重叠明显增加、双方都能持续获利，先把市场重新判断为宽通道、交易区间或转换阶段，不要强行套用趋势延续策略。

## 怎样选择具体策略

| 当时看到的过程 | 使用的策略 |
| --- | --- |
| 强趋势建立后第一次短而浅的回调 | [强趋势中的第一次小回调](first_small_pullback.md) |
| 第一次恢复尝试没有走远，回调再次发展后出现第二次恢复 | [趋势中的第二次恢复入场](second_entry_continuation.md) |
| 反趋势回调清楚形成三次推动，但主要趋势结构仍完整 | [趋势中的三推或楔形回调](wedge_pullback_continuation.md) |
| 牛趋势第一次深回调穿过均线，整根 K 线位于均线下方；熊趋势镜像 | [强趋势第一次均线缺口回调](first_ma_gap_pullback.md) |
| 价格连续二十根完整 K 线没有触及均线，随后第一次触及 | [长时间不触均线后的第一次测试](first_ma_test_after_long_gap.md) |
| 首次均线缺口棒后，第一次朝均线反转未触及，价格再次远离，随后第二次朝均线反转 | [第二次均线缺口回调](second_ma_gap_pullback.md) |

## 同一次回调只选择一套计划

同一段回调可能同时被描述成 H2、双底、楔形或均线测试，但只能选择最符合当时过程的一张策略页。计划写定后，不因后来出现更漂亮的名称而切换入场、缩小止损或延长目标。

如果原计划在成交前失效，先取消并确认订单已经结束；只有新的价格过程独立满足另一张策略页时，才重新建立计划。如果已经成交，先按原计划处理真实仓位，不能把策略改名来逃避止损。

## 所有趋势策略共同遵守

- 做多与做空使用完全镜像的判断；
- 入场以前就写清入场、撤单、止损、目标、数量和管理；
- 本目录的基线计划都只使用一次入场、一次全量目标，不加仓、不分批退出、不留尾仓；
- 目标已经被入场价越过，或途中存在更近且无法合理穿越的强障碍时，放弃；
- 看不清趋势、回调顺序或主要摆动点时，放弃；
- 订单、成交、真实仓位和保护操作统一遵守[人工交易执行手册](../../execution/execution_manual.md)。
