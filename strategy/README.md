# 人工策略手册

> **定位：供交易者根据当前市场选择一项完整策略**
>
> 本目录只回答“看到什么、什么时候行动、什么时候放弃以及怎样管理”。基础概念回到[核心框架](../core/README.md)，订单和成交安全回到[执行手册](../execution/execution_manual.md)。

策略页底部的“概念关联”只标出该页直接引用的定义，不构成缩减版理解清单。使用任何策略前，都应完整理解 `core/` 的全部组成及其关系。

## 先选择市场家族

选择策略前，先用当时可见的信息回答：

1. 基础状态更接近趋势还是交易区间，还是暂时看不清？
2. 如果是趋势，当前更像获接受的 breakout phase（spike）还是 channel；机会来自趋势内回调，还是成熟趋势的控制权转移？
3. 如果是区间，当前是边缘交易、双向等待还是已经出现突破过程？区间中部通常不选择区间反转策略。
4. 当前是否处于开盘或尾盘覆盖窗口，因而必须先检查时段页特有的截止时间、目标和退出要求？
5. 前方空间、合理止损和成本是否允许形成一份完整计划？

看不清时不强行分类。完整选择过程见[如何选择策略](how_to_choose.md)。

## 时段页是覆盖层，不是规避入口

趋势与交易区间是基础状态；获接受的 breakout phase（spike）或 channel 细化趋势阶段，breakout mode、双向等待和趋势反转过程则作为叠加信息或控制权转移线索。开盘与尾盘页在这些判断之上增加时间、目标和剩余风险约束。进入预先写定的开盘或尾盘窗口时，先检查[开盘与尾盘覆盖层](opening/README.md)，再决定当前机会是否还能进入普通策略家族。

- 同一个底层交易机会同时落入时段页和普通页时，使用时段页。“同一个”指普通页拟议入场与时段候选依赖同一参照事件、预先边界和方向；即使等待的信号或入场时点不同，仍是同一机会。时段页的截止时间、目标和退出要求优先，不得选择普通趋势、突破、区间或反转页绕过这些约束。
- 时段覆盖从具体页面规定的候选起点开始，只持续到该候选或计划被拒绝、失效、过期、未成交或结束；仅仅处于开盘或尾盘窗口，不会锁定无关的普通机会。一次只保留一项活动候选或计划。同一底层机会若同时满足两张时段页，而主要交易前提无法唯一确定，就继续等待或放弃，不用更有利的名称替自己选择。
- 时段候选或计划被拒绝、失效、过期、未成交或结束后，从当前价格重新判断。原信号、原触发和原订单不能改名或复用；新候选必须出现旧计划结束后才完成的当前结构或触发，并独立满足所选页面全部条件。已经完成的突破、接受、趋势或区间结构可以作为页面明确要求的历史背景。
- 时段页条件不完整或过期不会自动生成普通策略。若拟议普通入场仍依赖原时段候选的参照事件、边界和方向，却没有在旧候选结束后形成新结构或新触发，就直接放弃；只有新的当前机会独立合格时，才按普通页处理。

具体判断边界见[如何选择策略：时段策略怎样覆盖普通策略](how_to_choose.md#时段策略怎样覆盖普通策略)。

## 策略共同基线

[策略共同规则](common_rules.md)是页面基线目标、实际退出价格、计划冻结、净 R、等待失效和仓库管理分支的唯一权威位置。全部正式策略页继承这套基线；具体策略页只有明确写出时才可采用更严格要求或局部覆盖。

仓库选择结构保护止损和单一实际全量退出价格作为固定管理分支。价格行为概念回到 `core/`，订单回执、成交、仓位与保护等安全事实始终回到[执行手册](../execution/execution_manual.md)，不受策略层覆盖。

## 同一次机会只选择一项

相邻策略可能描述同一段价格运动的不同进入时点。例如一轮突破可能先出现强突破收盘，随后出现跟进，再出现第一次回测。它们是三种不同计划，不是三次自动加仓理由。

在机会开始前或第一项条件出现时，写清准备等待哪一种进入方式。已有活动计划时，不得因为后续又出现另一个策略名称而重复承担风险。若原计划结束，必须从当前价格重新检查另一项策略的全部条件。

原计划失败也不自动构成反手理由，见[原计划失败后如何重新判断](switching_after_failure.md)。

## 每个策略页都应回答什么

每一页只写一套完整计划，并明确：

- 交易的市场行为；
- 适用和不适用环境；
- 机会形成顺序；
- 条件完整与放弃边界；
- 入场、撤单、保护止损、目标和持仓管理；
- 成交后的正常波动与前提失效；
- 策略特有的订单风险；
- 正例、反例、边界案例和复盘问题。

这些页面是供人执行的仓库整理。执行前仍须设置具体市场、周期、时段、成本、账户和个人风险边界。

页面中的固定根数、观察窗口、入场时点和管理方法，是本仓库为了让同一规则能够一致执行而作出的人工基线选择，不应冒充 Brooks 适用于所有市场和周期的通用硬规则。
