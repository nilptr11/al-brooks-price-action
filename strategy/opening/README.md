# 开盘与尾盘策略

> **定位：供交易者处理开盘定价、早期区间和尾盘开盘价磁吸**
>
> 本目录是覆盖趋势、区间、突破和反转行为的时段层。时段不替代价格行为，但会增加截止时间、目标和强制退出要求。每个市场的交易日、开盘时点、半日交易和时区必须在一轮执行开始前固定；最终日型只能在复盘中填写。

## 怎样选择

| 当时能够确认的过程 | 只选择这一项 |
| --- | --- |
| 开盘首次测试前一交易日高点或低点，重新进入旧区间并形成二次反转 | [开盘测试前一交易日极值后的二次反转](opening_previous_extreme_reversal.md) |
| 前三十分钟形成开盘区间，随后突破收在边界外；跟进影线可以回测区间，但收盘仍在外侧并继续推进 | [开盘区间突破并获得接受](opening_range_breakout.md) |
| 开盘很快形成单向趋势，第一次完整回调后恢复 | [开盘趋势中的第一次回调](opening_trend_first_pullback.md) |
| 最后九十分钟仍是成熟区间，价格从外侧边缘拒绝并朝开盘价返回 | [尾盘区间向开盘价回归](late_session_open_magnet.md) |

## 时间和机会边界

- 前一交易日高低点反转的二次信号与入场单有效确认，都必须严格早于开盘后六十分钟截止时点。
- 开盘区间策略把开盘后前三十分钟固定为区间建立窗口；突破、跟进与入场单有效确认都必须严格早于开盘后九十分钟截止时点。
- 开盘趋势第一次回调、顺势信号与入场单有效确认，都必须严格早于开盘后六十分钟截止时点。
- 尾盘回归只在正常交易时段最后九十分钟考虑，并在正常收盘前一个完整观察周期开始强制退出流程。

“严格早于”按完成 K 线的收盘时间和平台有效确认时间判断。K 线恰好在截止时点收盘，或订单到截止时点才确认有效，都已经太晚；未成交订单在截止时点撤销。尾盘强制退出流程开始时点也必须在本轮开始前换算成明确时钟时间。

## 覆盖普通策略的边界

进入上述固定窗口后，先检查本目录。覆盖发生在活动时段候选与普通页拟议入场依赖同一参照事件、预先边界和方向时；即使信号或入场时点不同，也属于同一底层机会。覆盖只持续到候选或计划结束，一次早期触碰不会永久锁定之后所有价格行为。

| 时段页 | 同一当前机会可能重叠的普通页 |
| --- | --- |
| 前一交易日极值反转 | 普通趋势反转或区间边缘反转 |
| 开盘区间突破 | 普通强突破或突破跟进 |
| 开盘趋势第一次回调 | 普通趋势第一次小回调 |
| 尾盘开盘价回归 | 普通区间边缘确认 |

- 每张时段页规定的候选起点出现后，该候选才开始活动；仅仅进入时间窗口不会覆盖与四张页面无关的普通机会。候选活动期间，来自同一参照事件、预先边界和方向的普通入场仍是同一底层机会，即使它使用更早或更晚的信号。此时使用时段页，不能用普通页省略后续确认，也不能借普通页绕过六十或九十分钟截止、替换本页目标或取消尾盘强制退出。
- 同一底层机会如果同时满足两张时段页，而主要交易前提无法唯一确定，就继续等待或放弃；不能挑选限制较少的一页。活动候选、订单或持仓存在时，也不建立第二份计划。
- 时段候选被拒绝、失效、过期、未成交或结束后，从当前价格重新判断。原信号、原入场依据和原订单不能重新包装；新候选必须包含旧计划结束后才完成的当前结构或触发，独立满足所选页面全部条件，并重新计算入场、止损、目标、数量和剩余时间。
- 时段页条件不完整或太晚，不会自动授权普通页。若普通页仍依赖原时段候选的参照事件、边界和方向，且旧候选结束后没有形成新结构或新触发，就直接放弃；只有后来形成的全新当前机会才可重新评估。已经完成的突破、接受、趋势或区间结构只能作为新策略明确要求的历史背景。
- 当前机会与四张时段页均不重叠时，普通策略可以独立使用；仍须检查本轮结束时间、剩余空间和风险边界。

开盘区间突破计划未成交过期后，如果此前从未回测固定边界，后来发生的第一次边界触及是新的当前参照事件，可以从当前价格检查[突破后的第一次回测](../breakout/breakout_first_pullback.md)。原突破和跟进只作为该策略要求的历史背景；新的首次触及、恢复信号、入场、止损和目标必须重新形成。若旧计划结束前已经触及过边界，后来测试不再是第一次，不能使用这一转换。

同一开盘运动只选择一项策略。开盘区间突破、开盘趋势第一次回调和前一交易日极值反转可能先后出现在同一段走势中，但不是重复入场或加仓理由。旧计划尚有活动订单或持仓时，不建立另一项计划，无论新名称来自本目录还是普通策略家族。完整选择顺序见[如何选择策略](../how_to_choose.md#时段策略怎样覆盖普通策略)。

## 共同纪律

- 开盘价、前一交易日高低点、开盘区间窗口和正常收盘时点都要事先写清。
- 每页只使用一个入场方式、一个保护位置和一个全量目标。
- 开盘价、前一交易日极值和开盘区间量度都是本目录使用的可见 magnet 或投射来源；各页把其中一个固定为时段计划的基线目标，不表示 Brooks 认为该价格必然到达，也不得在成交后换成更远的日型目标。
- 不加仓、不分批退出、不留尾仓；到达策略规定的时间边界就撤单或退出。
- Opening gap 只表示重新定价，不单独决定方向。
- 最终趋势日、区间日或反转日不得参与当时判断。

共同选择流程见[如何选择并锁定一项策略](../how_to_choose.md)，执行环境和平台安全见[执行与复盘](../../execution/README.md)。
