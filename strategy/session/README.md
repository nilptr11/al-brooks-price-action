# 时段策略：开盘与尾盘

> **定位：供交易者处理开盘定价、开盘初段结构和尾盘开盘价磁吸**
>
> 本目录是覆盖趋势、区间、突破和反转行为的时段层。时段不替代价格行为，但会增加截止时间、目标和强制退出要求。每个市场的交易日、开盘时点、半日交易和时区必须在一轮执行开始前固定；最终日型只能在复盘中填写。

**来源层**：Brooks 课程支持开盘定价、opening reversal、opening-range / breakout-mode、trend from the open、尾盘时间风险和 session open magnet 等市场行为；本目录的 30/60/90 分钟窗口、根数、K 线比例、订单寿命、唯一目标与全量退出，是仓库为一致执行和复盘设定的时钟与管理基线，不是 Brooks 的统一参数，也不定义 Brooks 术语的最低边界。

本页只负责[全局选择状态机](../how_to_choose.md)委托的时段覆盖层内部分类，以及各时段候选对应的普通价格行为家族；具体资格、触发、参数和特有失效以所选策略页为准。本页不另定义覆盖生效、单候选、结束或重入规则。

## 开盘对象只用于路由

课程中的开盘对象不共享一个固定窗口。本目录只用它们说明应继续检查哪一类现有策略，不把名称本身当作 setup、entry 或新增策略：

| 开盘对象 | 本目录采用的边界 | 路由方式 |
| --- | --- | --- |
| Opening / first swing | 从 session 开始到第一段向上或向下 swing 结束的描述性阶段；终点通常要等后续价格走出后才清楚，不是实时可预知的固定 K 线 | 只作为开盘背景；当时实际出现前一交易日极值测试、开盘趋势第一次回调或其他完整过程时，再检查下方已有页面或普通策略家族 |
| 宽泛 Opening Breakout Mode（Opening BOM） | 可以包括开盘强突破后很快形成早期区间、等待下一次方向选择；课程中“首根 K 线后两侧先后受到测试和反转”的较成熟子型仍使用随早期价格更新的动态边界，两者都不等同于固定前三十分钟范围 | 尚未选择方向时回到普通 trading-range / breakout-mode 判断；只有实际突破、跟进和接受出现后才路由到突破家族，不在本目录增设一张通用 BOM 策略 |
| First-18-bar heuristic | 课程针对相应日内图表观察前 18 根范围及其后附近突破或反转的近似启发式；不是 Opening BOM 的同义词，也不是通用阈值 | 本目录没有以 18 根为边界的策略；除非另有经品种和周期验证的正式页面，否则它只保留为观察提示 |
| 仓库固定窗口 | 各具体页为了可重复执行预先声明的 30、60 或 90 分钟时钟 | 只约束写出该窗口的页面：前三十分钟只定义固定范围突破页，六十／九十分钟只定义相应截止或尾盘扫描窗，不能反推为 Brooks 的 first swing、Opening BOM 或 first-18 定义 |

Opening Reversal 同样只是早段原方向在已知位置失去接受后的过程名。只有下方既有页面的完整顺序、风险和时间条件同时成立时才获得执行资格；若相同结构到较晚阶段才形成，应回到普通反转或 MTR 家族判断，不能继续借用开盘标签。

## 怎样选择

| 当时能够确认的过程 | 只选择这一项 |
| --- | --- |
| 开盘首次测试前一交易日高点或低点，重新进入旧区间并形成二次反转 | [开盘测试前一交易日极值后的二次反转](opening_previous_extreme_reversal.md) |
| 仓库把前三十分钟冻结为固定范围，随后突破收在边界外；跟进影线可以回测范围，但收盘仍在外侧并继续推进 | [固定前三十分钟范围突破并获得接受](opening_range_breakout.md) |
| 开盘很快形成单向趋势，第一次完整回调后恢复 | [开盘趋势中的第一次回调](opening_trend_first_pullback.md) |
| 最后九十分钟仍是成熟区间，价格从外侧边缘拒绝并朝开盘价返回 | [尾盘区间向开盘价回归](late_session_open_magnet.md) |

## 时间和机会边界

- 前一交易日高低点反转的二次信号与入场单有效确认，都必须严格早于开盘后六十分钟截止时点。
- 固定前三十分钟范围策略把开盘后前三十分钟作为建立窗口；突破、跟进与入场单有效确认都必须严格早于开盘后九十分钟截止时点。
- 开盘趋势第一次回调、顺势信号与入场单有效确认，都必须严格早于开盘后六十分钟截止时点。
- 尾盘回归只在正常交易时段最后九十分钟考虑，并在正常收盘前一个完整观察周期开始强制退出流程。

“严格早于”按完成 K 线的收盘时间和平台有效确认时间判断。K 线恰好在截止时点收盘，或订单到截止时点才确认有效，都已经太晚；未成交订单在截止时点撤销。尾盘强制退出流程开始时点也必须在本轮开始前换算成明确时钟时间。

## 覆盖普通策略的边界

候选何时活动、同一底层机会如何覆盖普通策略、旧计划结束后怎样重新形成新机会，统一由[如何选择并锁定一项策略](../how_to_choose.md#时段策略怎样覆盖普通策略)定义；本页不维护第二套状态机。以下表格只说明本目录四张时段页可能覆盖哪个普通家族：

| 时段页 | 同一当前机会可能重叠的普通页 |
| --- | --- |
| 前一交易日极值反转 | 普通趋势反转或区间边缘反转 |
| 固定前三十分钟范围突破 | 普通强突破或突破跟进 |
| 开盘趋势第一次回调 | 普通趋势第一次小回调 |
| 尾盘开盘价回归 | 普通区间边缘确认 |

同一时段运动只选择一项活动策略。固定前三十分钟范围突破、开盘趋势第一次回调、前一交易日极值反转与尾盘回归可以先后出现在同一交易日，但不是重复入场或加仓理由；旧计划结束前不建立第二份风险。

## 时段覆盖特有纪律

价格、净 R、等待失效、保护与固定管理的通用语义只由[策略共同规则](../common_rules.md)定义。下列内容只补充开盘与尾盘覆盖层：

- 开盘价、前一交易日高低点、开盘区间窗口和正常收盘时点都要事先写清。
- 开盘价、前一交易日极值和开盘区间量度都是本目录使用的可见 magnet 或投射来源；各页把其中一个固定为时段计划的基线目标，不表示 Brooks 认为该价格必然到达，也不得在成交后换成更远的日型目标。
- 到达具体策略规定的时间边界就撤单或退出；时段边界是本家族对共同基线的显式覆盖。
- Opening gap 只表示重新定价，不单独决定方向。
- Opening / first swing、Opening BOM、first-18 heuristic 和仓库固定窗口只负责区分观察对象与后续路由，任何一个名称都不单独授权下单。
- 最终趋势日、区间日或反转日不得参与当时判断。

共同选择流程见[如何选择并锁定一项策略](../how_to_choose.md)，执行环境和平台安全见[执行与复盘](../../execution/README.md)。
