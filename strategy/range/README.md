# 交易区间策略

> **定位：供交易者在成熟宽区间的边缘选择一种反转进入方式**

## 什么时候进入这个家族

交易区间策略依赖旧公平区域继续主导。最低背景是：

- 价格已经多次从上下两侧反转，而不是刚出现第一次横向停顿；
- 双向 K 线和重叠增多，至少多次穿越区间中部；
- 上沿、下沿和中部能够在机会出现前画出；
- 当前位于外侧边缘，而不是区间中部；
- 边缘到现实目标之间仍有足够空间。

区间内可以出现很强的单腿。只有新价格在边界外获得收盘、跟进和持续接受，才离开本家族转向突破策略。

## 三项策略怎样选择

| 当前路径 | 只选择这一项 |
| --- | --- |
| 尚未发生真实突破，边缘出现明确反向信号 | [成熟区间边缘确认后反转](range_edge_confirmation.md) |
| 尚未发生真实突破，事先决定以较小首仓和两层限价处理宽区间 | [成熟区间边缘限价试仓与预设加仓](range_edge_limit_scale_in.md) |
| 已经收在边界外，随后重新进入旧区间 | [突破失败后回到区间](failed_breakout_return.md) |

确认入场和限价加仓的成交价格、概率、止损和仓位逻辑不同，同一次边缘测试只能选择一种。真实突破发生后，也不能继续把它当作普通边缘触碰。

## 扩张三角形怎样处理

扩张三角形是高点更高、低点更低的波动扩大型区间，不是压缩结构。只有外侧摆动能够事先画定、区间仍保持双向交易并且目标空间合理时，才使用本家族：

- 外侧边缘出现反向确认，使用边缘确认策略；
- 事先具备宽止损和两层计划，使用限价试仓策略；
- 越界收盘后重新进入，使用失败突破策略；
- 突破获得接受，离开本家族并重新判断突破策略。

## 什么时候离开这个家族

- 区间还不成熟，边界只能事后画出；
- 当前位于区间中部；
- 区间太紧，扣除成本后没有现实利润；
- 边界外出现强突破和跟进；
- 合理结构止损超出本次风险边界；
- 大周期重要位置使原区间目标失去意义。
