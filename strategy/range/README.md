# 交易区间策略

> **定位：供交易者在成熟宽区间的边缘选择一种反转进入方式**

## 什么时候进入这个家族

交易区间策略依赖旧公平区域继续主导。最低背景是：

- 价格已经多次从上下两侧反转，而不是刚出现第一次横向停顿；
- 双向 K 线和重叠增多，至少多次穿越区间中部；
- 上沿、下沿和中部能够在机会出现前画出；
- 当前位于外侧边缘，而不是区间中部；
- 边缘到按本目录公式固定的中部目标之间，净 R 达到本轮预先写定门槛。

区间内可以出现很强的单腿。只有新价格在边界外获得收盘、跟进和持续接受，才离开本家族转向突破策略。

## 固定边界与区间中部

三张区间策略共用同一个中部算法：机会成为候选以前，把成熟区间上沿记为一个固定可交易价格、下沿记为另一个固定可交易价格；原始中部等于“上沿加下沿再除以二”。原始中部不在最小跳动上时，目标向计划入场一侧取整，使目标更近而不是更远：下沿做多向下取到最近可交易价格，上沿做空向上取到最近可交易价格。

候选形成以后，上沿、下沿和中部全部冻结。本轮不得把“中部”解释成中央价值区、视觉中心、区间中间三分之一中的任意价格，也不得因后来扩张重新计算。固定中部不能提供本轮最低净 R，或其前方有使目标不可合理到达的强障碍时，放弃而不是更换目标。

精确算术中点是本仓库为了让区间边缘向内 scalp 得到唯一可挂单价格而选择的保守基线，不是 Brooks 对所有 trading-range fade 规定的统一目标。Brooks 的上层原则是 BLSHS、避开中部入场，并根据区间宽度、概率、止损和途中磁铁选择现实利润；本目录只固定其中一种管理方案，不把中点到达写成保证。

## 三项策略怎样选择

| 当前路径 | 只选择这一项 |
| --- | --- |
| 尚未出现边界外收盘，边缘出现明确反向信号 | [成熟区间边缘确认后反转](range_edge_confirmation.md) |
| 尚未出现边界外收盘，事先决定以较小首仓和两层限价处理宽区间 | [成熟区间边缘限价试仓与预设加仓](range_edge_limit_scale_in.md) |
| 已经收在边界外，随后重新进入旧区间 | [突破失败后回到区间](failed_breakout_return.md) |

确认入场和限价加仓的成交价格、概率、止损和仓位逻辑不同，同一次边缘测试只能选择一种。边界外收盘或持续接受出现后，也不能继续把它当作普通边缘触碰。

## 扩张三角形怎样处理

扩张三角形是高点更高、低点更低的波动扩大型区间，不是压缩结构。只有外侧摆动能够事先画定、区间仍保持双向交易并且目标空间合理时，才使用本家族：

- 外侧边缘出现反向确认，使用边缘确认策略；
- 事先具备宽止损和两层计划，使用限价试仓策略；
- 越界收盘后重新进入，使用失败突破策略；
- 突破获得接受，离开本家族并重新判断突破策略。

## 什么时候离开这个家族

- 区间还不成熟，边界只能事后画出；
- 当前位于区间中部；
- 区间太紧，扣除成本后没有现实利润；
- 边界外出现强突破和跟进；
- 合理结构止损超出本次风险边界；
- 大周期重要位置使原区间目标失去意义。
