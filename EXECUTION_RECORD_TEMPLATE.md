# 执行记录模板

> **状态：Execution / Derived**
>
> 本文件只提供可复制的批次配置和单笔交易空白记录；价格行为判断回到相应学习内容与来源，仓库订单和记录操作见当前[执行手册](EXECUTION_MANUAL.md#7-仓库订单与记录操作)。本模板不增加 setup、stop、target 或 Trader's Equation 规则。

## 使用方式

- 执行环境或共同假设发生实质变化时，复制一份新的[批次执行配置](#批次执行配置)并分配新 ID。
- 准备把一个 setup 组织成包含 entry、stop、target 和 Trader's Equation 的交易时，复制一份[单笔交易记录](#单笔交易记录)；主动放弃、撤单或最终未成交也可以保留结束原因。
- 原始计划与后续事件的保存方式按[仓库订单与记录操作](EXECUTION_MANUAL.md#7-仓库订单与记录操作)处理。
- 连续回放、其他模拟和实盘使用同一版式，通过配置中的执行来源区分。
