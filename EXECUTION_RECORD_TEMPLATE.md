# 执行记录模板

> **状态：Execution / Derived**
>
> 本文件只提供可复制的批次配置和单笔交易空白记录；所有判断、准入、订单与风险规则以当前[执行手册](EXECUTION_MANUAL.md)为准。发生冲突时修改本模板以服从手册，不在此新增执行规则。

## 使用方式

- 执行环境或共同假设发生实质变化时，复制一份新的[批次执行配置](#批次执行配置)并分配新 ID。
- 自然出现值得计算 entry、stop 和 target 的候选时，复制一份[单笔交易记录](#单笔交易记录)；数学不合格、主动放弃或最终未成交也保留结束原因。
- 复制后记录实例的冻结、结束、重建与追加方式按[执行流程](EXECUTION_MANUAL.md#2-一条执行流程)处理。
- 连续回放、其他模拟和实盘使用同一版式，通过配置中的执行来源区分。
