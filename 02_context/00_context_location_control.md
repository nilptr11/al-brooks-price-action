# Context、位置和控制权

## 它解决什么问题

Context 决定一个形态是有意义的交易结构，还是普通噪声。Brooks 语境中，context 通常比 signal 更重要。

## 控制权

控制权描述当前多空谁更主动。控制差描述优势是否足够明显。

控制权来自价格行为，而不是主观观点：

- 强收盘。
- 连续性。
- 反方失败。
- 突破后的跟进。
- 回调质量。
- 是否快速收回关键区域。

控制权强时，顺势交易更合理。控制权弱时，应更多关注区间边缘、失败突破和被困交易者。

## 位置

位置回答“价格在哪里”。同一个 signal bar 出现在不同位置，含义可能完全不同。

重要位置包括：

- 前高和前低。
- 交易区间上沿、下沿和中轴。
- 突破点和回探位置。
- 缺口。
- 量度目标。
- 大周期支撑阻力。
- 大整数位和视觉价位。

位置本身不交易，位置加行为才形成交易意义。

## 支撑阻力不是线

支撑阻力是交易者会改变行为的区域，不是精确价格。价格可能略微突破后才失败，也可能提前反应。

到达重要区域后要观察：

- 是否加速突破？
- 是否停顿并形成双边交易？
- 是否突破后快速收回？
- 是否回探后守住？
- 哪一边被困？
